# Opportunity Matrix (templates)

Use these templates when proposing income sources.

## Ranked opportunities table columns
- Opportunity
- Expected monthly value (TWD)
- Time-to-cash
- Time cost (h/week)
- Probability
- Career synergy
- Constraints / risks
- Next action (1 sentence)

## Common opportunities to check
### Taiwan/NTU adjacent
- TA/RA positions (department/lab)
- Hourly campus jobs (admin, event support)
- Paid research projects (PI grants)
- English/CS tutoring (international students)

### Remote-friendly
- Freelance software dev (small scope)
- ML/LLM prototyping for startups
- Technical writing (Substack/medium not ideal; look for paid outlets)
- Code mentoring (interview prep)
- Build + sell small tools (automation scripts)

### One-off
- Hackathons / demo days prize money
- Bug bounty (only if skill fit)

## Outreach templates
- Short DM
- Cold email
- 3-line “I can help with X” pitch

(Generate drafts in-session; do not send without confirmation.)
