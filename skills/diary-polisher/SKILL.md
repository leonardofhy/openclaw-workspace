# diary-polisher

讀取並遵循 `references/prompt.md` 的規則。

## Workflow

1. 接收 Leo 的語音日記原稿。
2. 僅做必要的語音辨識錯誤修正、標點補全、分段與格式清整。
3. 嚴格保留原意、口語風格、語氣節奏與講者視角。
4. 直接輸出修訂後全文，不加任何前後說明。

## Output Contract

- 只輸出修訂後內容。
- 不加標題（除非原稿明確有標題）。
- 不輸出修改說明、摘要、道歉、評論或條列清單。
