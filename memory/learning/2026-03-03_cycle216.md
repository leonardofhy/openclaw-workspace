## KEY FIND: AG-REPA (arXiv:2603.01006, ICML submission)

**"Attribution-Guided REPresentation Alignment for Audio Flow Matching"**

**Core concept: Store-Contribute Dissociation (SCD)**
- Layers with highest representational similarity to semantic/acoustic teacher features ≠ layers that causally contribute most to audio generation
- Deep layers = "semantic reservoirs" (high LASP teacher-space similarity score)
- Early/mid layers = "causal drivers" (high FoG-A attribution score, Jacobian sensitivity)
- Dissociation is spatiotemporal: mid-layer (L6-12) become causal drivers during intermediate denoising phase (t≈0.5)

**FoG-A (Forward-only Gate Ablation)**
- Each layer's causal contribution = induced change in predicted velocity field when that layer is ablated
- Forward-only = efficient (no backprop), just measure output change
- Pearl Level 2 (interventional gate ablation), NOT Level 3 (counterfactual IIT)

**Not a competitor:** Audio generation (Flow Matching TTS+TTA) ≠ speech LLM understanding/interpretability. Leo's task is understanding how speech LLMs process audio representations.

## Paper A Relevance

SCD directly motivates the Listen Layer Hypothesis for speech LLMs:
- AudioLens: probes representational similarity (= stores what) → Pearl Level 1
- AG-REPA FoG-A: tests causal contribution (= causes what) → Pearl Level 2 (generation)
- Paper A DAS-IIT: counterfactual intervention on speech LLM behavior (= causally locates Listen Layer) → Pearl Level 3
