### Key insight from abstract (v4, May 2025, 135KB):
The paper makes 3 contributions:
1. **Generalizes causal abstraction**: from mechanism *replacement* (hard/soft interventions) to arbitrary mechanism *transformation* (functionals old→new). This is the theoretical move that allows DAS (rotate-fix-unrotate) to be formally correct.
2. **Formalizes core MI concepts**: polysemantic neurons, linear representation hypothesis, modular features, graded faithfulness — all in the same language.
3. **Unification**: DAS, SAEs, activation patching, path patching, causal tracing, concept erasure, circuit analysis, causal mediation analysis, causal scrubbing, differential binary masking, steering = ALL special cases of causal abstraction.
