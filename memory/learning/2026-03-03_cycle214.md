### Key Find: EG-GRVQ (arXiv:2603.01476) — Entropy-Guided Codec

**Core insight:** "Assuming channel activations follow approximately Gaussian statistics, the **variance of each channel** can serve as a principled proxy for its information content."

This independently validates TWO things in Leo's research:

1. **Gap #21 (RVQ Layer 1 = semantic):** EG-GRVQ uses entropy-guided grouping to partition encoder output into semantic and acoustic branches. The "variance per channel" proxy is EXACTLY the same principle as SpeechTokenizer's Layer 1 = semantic / Layers 2+ = acoustic split. The empirical finding = a second independent data point confirming that codec channel information content is stratified by variance.

2. **whisper_hook_demo.py norm heatmap (cycle #12 + Asiaee 2602.24266 theoretical justification):** The norm heatmap uses activation variance across layers as a pre-screen for causally relevant layers. EG-GRVQ now provides EMPIRICAL CODEC evidence that variance = information content in speech encoders. This tightens the theoretical + empirical triangle: Asiaee (theory) + EG-GRVQ (codec empirics) + whisper_hook_demo.py (Whisper application).

**Connection to Paper A:** When designing "corrupt" stimuli using RVQ, selecting which codec tokens to perturb based on channel variance (EG-GRVQ method) = principled clean/corrupt split. This gives a third independent validation (beyond SpeechTokenizer and DashengTokenizer) that semantic content clusters in high-variance channels.

**Note:** EG-GRVQ is NOT a competitor — it's codec engineering, not speech LM interpretability. But it's a citable empirical prior for the "variance = information content" claim that underpins both whisper_hook_demo.py and the RVQ clean/corrupt protocol.

### KEY FIND: EmbedLens (arXiv:2603.00510, CVPR 2026) — Visual Token Sparsity

**Fan et al.** introduce **EmbedLens**, a probing tool for visual token analysis in MLLMs.

**Core findings:**
1. Visual tokens partition into **sink** (attention sinks), **dead** (carry no image info), and **alive** (~60% — carry image-specific meaning)
2. **Mid-layer injection is sufficient**: alive visual tokens naturally align with intermediate LLM layers; shallow-layer processing is redundant
3. EmbedLens is observational (probing) = Pearl Level 1

**Relevance to Paper A "Listen Layer" for speech LLMs:**

| Finding | Paper A Speech Analog |
|---------|----------------------|
| Only ~60% of visual tokens are "alive" | Not all audio tokens carry speech content (silence frames, padding = "dead" audio tokens) |
| Mid-layer injection of visual tokens is optimal | Listen Layer hypothesis: audio representations causally consulted at one intermediate LLM layer |
| Shallow layers = redundant for vision | AudioLens: audio grounding starts late (layers 6+ in AudioLens observations) |
| EmbedLens = observational | Leo's DAS-IIT = causal (Pearl Level 3) |

**Key difference from Paper A**: EmbedLens is vision-only, observational (Level 1). Leo = speech + causal (Level 3). The structural analogy strengthens motivation ("VLMs have dead/alive visual tokens + mid-layer alignment — do speech LLMs have the same structure, and where is the Listen Layer?")

**Action**: Add EmbedLens to Paper A related work table as Level 1 vision analog. The CVPR 2026 acceptance validates the "mid-layer injection" framing in vision → motivates Leo to find the speech equivalent causally.

**Note on audio token taxonomy**: Could add preprocessing step to Paper A — classify audio tokens into sink/dead/alive using channel variance (EG-GRVQ method) BEFORE DAS patching → focus DAS on alive tokens only → cleaner causal signal + efficiency gain.
