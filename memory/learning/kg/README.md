# Knowledge Graph Directory

- `../knowledge-graph.md` is the current source of truth for accumulated research knowledge
- Future autodidact cycles should write new knowledge entries as individual .md files in this directory
- `kg_query.py` (skills/autodidact/tools/) indexes both `kg/*.md` files AND `knowledge-graph.md`
- File naming: `{topic-slug}.md` (e.g., `and-or-gates.md`, `ravel-mdas.md`)
