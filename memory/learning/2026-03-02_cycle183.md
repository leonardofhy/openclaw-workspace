## Papers Scanned

### 1. Causal Abstraction: A Theoretical Foundation for MI (Geiger et al., arXiv:2301.04709)
- **v4 last updated: May 2025** — still active development
- **3 key contributions** from abstract:
  1. **Generalize** causal abstraction from hard/soft interventions → arbitrary mechanism transformation (functionals from old → new mechanisms)
  2. **Formalize** polysemantic neurons, linear representation hypothesis, modular features, graded faithfulness
  3. **Unify** all major MI methods under one language: activation patching, path patching, causal mediation, causal scrubbing, causal tracing, circuit analysis, concept erasure, SAEs, DAS, steering
- **Core idea (from Geiger 2021, NeurIPS):** Align neural representations with variables in interpretable causal model → use interchange interventions to verify that the neural representations have the causal properties of those variables
- **Interchange intervention (II):** Take activation from one input, patch it into another input's forward pass at aligned representation → if output changes as predicted by causal model, the alignment is verified
- **IIT connection:** Paper A's "Listen Layer" experiment uses interchange interventions on Whisper connector outputs to verify where audio-semantic alignment occurs

### 2. The Non-Linear Representation Dilemma (Sutter, Minder, Hofmann, Pimentel — NeurIPS 2025 Spotlight, arXiv:2507.08802)
- **Critical theoretical finding:** Under arbitrary (non-linear) alignment maps, **any neural network can be mapped to any algorithm** → causal abstraction becomes trivial and uninformative
- **Proof:** With sufficiently powerful alignment maps → 100% interchange-intervention accuracy on RANDOMLY INITIALIZED language models (IOI task)
- **Implication:** The linear representation hypothesis is not just an empirical convenience — it's a **necessary constraint** for causal abstraction to be meaningful
- **The dilemma:** If we lift linearity constraint → vacuous; if we keep it → might miss important non-linear structure
- **Gap they identify:** Need principled way to balance alignment map complexity vs. accuracy; study connection between encoding assumptions and causal abstraction
