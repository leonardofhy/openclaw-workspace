### Deep Notes: 2 Relevant Papers

#### 1. DashengTokenizer (2602.23765) — Dinkel et al.
**"One layer is enough for unified audio understanding and generation"**
- Claim: frozen semantic features + acoustic injection from ONE additional layer outperforms conventional audio codecs+semantic encoders across 22 tasks
- Outperforms VAE-based methods on TTA/TTM; competitive on speech emotion, music, scene classification
- Inverts conventional paradigm: instead of (acoustic encoder → semantic injection), they do (frozen semantic → acoustic injection)
- **Connection to Track 3 / Listen Layer Hypothesis:**
  - If one semantic layer is sufficient for cross-task performance, finding WHICH LLM layer causally consults this single semantic layer becomes even more valuable
  - Supports the existence of a single "crystallization point" → consistent with Triple Convergence Hypothesis
  - Leo's Paper A locates this layer causally; DashengTokenizer shows one layer suffices behaviorally → convergent evidence
- **Connection to Track 1 / Benchmark Design:**
  - One-layer semantic tokenizer = cleanest possible clean/corrupt signal source for patching experiments
  - "Semantic layer only" corruptions = exactly the RVQ Layer 1 = content insight from Gap #21 (cycle #162-165)
- Verdict: SCAN (motivates, doesn't compete)

#### 2. FAD Encoder Bias (2602.23958) — Jeong et al., Interspeech 2026 submission
**"An Empirical Analysis of Task-Induced Encoder Bias in Fréchet Audio Distance"**
- Core finding: FAD scores depend heavily on encoder training task
  - AudioMAE → leads precision sensitivity
  - ASR Whisper → dominates structural detection but **blind to signal degradation**
  - VGGish (classification) → maximizes semantic detection but penalizes intra-class variation
- "Since no single encoder is a universal evaluator, future metrics must shift toward evaluation-native encoders intrinsically aligned with human perception"
- **Direct connection to Paper B (AudioSAEBench):**
  - Citable motivation for WHY AudioSAEBench needs multi-metric evaluation (not single encoder)
  - Whisper's structural bias but acoustic blindness = motivation for Category 3 (Reconstruction Fidelity) — SAE features should capture BOTH semantic AND acoustic dimensions
  - "Evaluation-native encoders aligned with human perception" = exactly AudioSAEBench's goal (SAE features as interpretable perceptual units)
  - Add to Paper B Related Work section: FAD encoder bias → AudioSAEBench addresses same problem at feature level
- Venue note: submitted to Interspeech 2026 (same deadline window as Leo's work) — not a competitor, a supporter
