**Question:** When Paper A sweeps gc(k) using DAS (RotatedSpaceIntervention in pyvene), the rotation matrices are learned by gradient descent over the intervene stimuli. Is there a false-positive risk if the learned rotation finds a subspace that correlates with the listen layer signal but is NOT the phonological subspace confirmed by Gap #18 (Choi et al.)?

**Why this matters:**
- DAS is unconstrained: the rotation matrix W âˆˆ R^{dÃ—k} is learned purely to maximize IIA
- If audio features are distributed (AudioSAE: 2000 features needed vs tens for text), W could pick up a spurious audio-correlated direction that isn't phonologically meaningful
- This would inflate gc(k) at layer L even if that layer isn't the "true" listen layer

**Analysis:**
1. **False positive scenario**: Layer L has high IIA because DAS finds a correlation with a confound (e.g., audio energy/volume level, which is trivially different between audio-dominant vs text-dominant stimuli in ALME)
2. **Guard against it**: Use ALME stimuli where audio-text *content* conflicts but audio *acoustic quality* is matched â†’ confounds controlled
3. **Better guard**: Cross-generalization test (A3 risk from Q14) â€” if the rotation learned on phoneme-contrast stimuli transfers to speaker-identity stimuli and emotional stimuli, the rotation is capturing something genuinely modality-grounded, not stimulus-specific

**Key insight (new):**
The Gap #18 experiment (Choi et al. phonological geometry) can serve as a **rotation initializer** for DAS, not just a prerequisite test:
- Instead of random initialization, initialize W = top-k PCA directions of the phonological geometry subspace from S3M encoder
- If DAS gc(k) with this initialization is higher than random init, it validates that the phonological subspace is the causally relevant one
- If similar, the learned rotation finds the same subspace regardless of init â†’ extra robustness evidence
