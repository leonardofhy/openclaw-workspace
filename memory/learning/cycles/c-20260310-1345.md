## Experiment Spec: Microgpt × RAVEL

### 1. Why This Matters

RAVEL (Rozanova et al.) measures whether a feature is **causally sufficient** (Cause) and **specific** (Isolate). We plan to apply AudioRAVEL to Whisper internals — but we can't verify correctness there (no ground truth). A microgpt gives us ground truth by construction.

**Claim we validate:** If RAVEL works correctly, Cause ≈ 1.0 and Isolate ≈ 1.0 on known circuits. Failure here reveals RAVEL's own assumptions (multi-layer routing, superposition, etc.).

---

### 2. Model: 30-Node Microgpt

Architecture (fully inspectable):
```
Input: phoneme embedding (6 phonemes × one-hot)
Layer 1: 30-node linear + ReLU
Layer 2: 30-node linear + ReLU
Output: binary category (voiced / unvoiced)
```

- **Size**: 6 × 30 + 30 × 30 + 30 × 2 = 180 + 900 + 60 = **1140 parameters**
- **Task**: Phoneme voicing classification (voiced: /b, d, g, v, z/ ; unvoiced: /p, t, k, f, s/)
- **Dataset**: Synthetic, 10k samples (phoneme → label, no real audio needed)
- **Training**: Adam, lr=1e-3, 200 epochs, CPU <30s

**Ground truth circuit by construction**:
- L1 nodes 0-14: learned to detect voicing feature (via weight inspection post-train)
- L2 nodes 0-4: aggregation of L1 voicing nodes
- We identify these by largest weight magnitudes to voicing-direction

---

### 3. RAVEL Protocol

We adapt RAVEL's intervention framework:

**Attribute pairs (2x2):**
- `voiced-yes` vs `voiced-no` (target attribute)
- `phoneme-labial` vs `phoneme-alveolar` (non-target; tests Isolate)

**For each L1 node j, compute:**
```python
Cause(j) = P(correct_voiced_output | intervene: set j = j_voiced_source)
           - P(correct | clean_forward)
# ≈ does this node carry causal signal for voicing?

Isolate(j) = 1 - P(place_changed | intervene: set j = j_voiced_source)
# ≈ does intervening on j NOT disturb unrelated attribute (place)?
```

**Expected results on known circuit nodes (L1 0-14):**
- Cause: 0.85–0.95 (high; these nodes do carry voicing)
- Isolate: 0.80–0.90 (high; voicing is separate from place in our toy data)

**Expected results on non-circuit nodes (L1 15-29):**
- Cause: <0.1
- Isolate: irrelevant

---

### 4. What Failure Modes Tell Us

| Observation | Interpretation |
|---|---|
| Cause low even on known nodes | RAVEL assumes linear effect; microgpt may route non-linearly |
| Isolate low on known nodes | Voicing/place co-occur even in synthetic data (check generation) |
| Both near random | Intervention scale or direction wrong — debug first |
| Known nodes = high, random nodes = low | ✅ RAVEL validated — proceed to Whisper |
