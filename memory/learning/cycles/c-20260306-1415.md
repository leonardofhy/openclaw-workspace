**Task**: Q051 | Track: T5 | Action: learn

## Problem / Context
OpenAI released GPT-5.4 — a new frontier model that integrates the best of GPT-5.3-Codex and GPT-5.2's reasoning into a single model. Released in ChatGPT (as GPT-5.4 Thinking), API, and Codex.

## Key Facts

### Model Capabilities
- **First general-purpose model with native computer-use** — agents can operate full computers across applications
- **1M token context** for long-horizon agentic tasks
- **Tool search** — helps agents find the right tool in large ecosystems without sacrificing intelligence
- **Most token-efficient reasoning model** — significantly fewer tokens vs GPT-5.2 → faster + cheaper

### Benchmarks (GPT-5.4 vs GPT-5.3-Codex vs GPT-5.2)
| Benchmark | GPT-5.4 | GPT-5.3-Codex | GPT-5.2 |
|-----------|---------|---------------|---------|
| GDPval (knowledge work) | **83.0%** | 70.9% | 70.9% |
| SWE-Bench Pro | **57.7%** | 56.8% | 55.6% |
| OSWorld-Verified (computer-use) | **75.0%** | 74.0% | 47.3% |
| Toolathlon | **54.6%** | 51.9% | 46.3% |
| BrowseComp | **82.7%** | 77.3% | 65.8% |

- On computer-use (OSWorld), GPT-5.4 jumps from 47.3% → 75.0% vs GPT-5.2 — massive leap
- **33% fewer false claims** and **18% fewer error-containing responses** vs GPT-5.2

### ChatGPT-Specific Features
- "Upfront plan of thinking" — user can steer mid-response before final output
- Better deep web research for specific queries
- GPT-5.4 Pro variant for max performance

## Connection to My Research

**Relevance to T3/T5 (listen-layer / gc(k)):**
- GPT-5.4 explicitly targets agentic audio/multimodal workflows → the *scale* of audio-LLM deployment is growing fast; safety work (T5) becomes more urgent
- Native computer-use + 1M context = longer reasoning chains → gc(k) confidence collapse signatures may be harder to detect (more distributed)
- OSWorld jump: 47.3% → 75.0% is a 27pp gain on computer-use. This is *grounding at scale* — exactly the question behind listen-vs-guess: does grounding quality track capability gains linearly?
- Hallucination reduction (33%) aligns with our hypothesis that better audio grounding = fewer confabulations; worth citing as motivating evidence
