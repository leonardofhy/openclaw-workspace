## Key Finding (important)

**The single-step SOT cross-attention proxy is not a valid AND-frac discriminator for beam rescoring.**

Root cause: At decoder step 1 (SOT token only), cross-attention weight concentration is:
- Driven by audio *energy* and spectral structure, not phoneme-specific evidence
- Noisy/accented audio has *higher* max attention (louder, more salient spectral features)
- This inverts the expected H3 relationship (accented should be LOWER AND-frac)

The Q166 steerability proxy worked because it measured *change* in AND-frac under gradient patching — relative, not absolute. For beam rescoring, we need AND-frac at *multiple decoder steps* where the model is actually committing to phonemes, not at SOT.

## Fix Required for Q172 Full Pass

Need to extend the decoder rollout to 5-10 tokens before measuring AND-frac. This requires:
1. Generating actual transcription hypothesis (run Whisper decode on audio)
2. Hooking cross-attention at each decode step during the transcription
3. AND-frac = mean max attn averaged over decode steps (not just SOT)

This is Tier 1 but requires generating real text (not just audio → encoder) — still CPU-doable.
