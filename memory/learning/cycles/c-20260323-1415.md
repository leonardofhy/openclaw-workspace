## L2-ARCTIC: Key Facts

**Structure:**
- 24 non-native English speakers × 6 L1 groups (4 speakers each: 2M + 2F)
- L1 groups: Arabic, Mandarin, Hindi, Korean, Spanish, Vietnamese
- ~1132 sentences per speaker (~1 hour read speech from CMU ARCTIC prompts)
- Forced-aligned phoneme transcriptions for ALL utterances
- Manual annotation: 150 utterances per speaker
  - 100 sentences (common across all speakers)
  - 50 sentences (L1-specific phonemes likely to be hard)
  - Annotation types: **substitution, deletion, addition** (mispronunciation tags)

**Dataset size:** 24 speakers × 1132 sentences ≈ 27,168 utterances total

## Known L1-Specific Phoneme Patterns (for mock grounding)

| L1 | Common substitutions | Key difficulties |
|----|---------------------|-----------------|
| Arabic | /p/→/b/, /v/→/f/ | pharyngeals, no /p/ |
| Mandarin | /l/↔/n/, /r/→/l/ | no consonant clusters, tonal transfer |
| Hindi | dental/alveolar confusion | retroflex insertion |
| Korean | /p,t,k/ as tense stops | epenthetic vowels, /l/↔/r/ |
| Spanish | /v/→/b/, /θ/→/s/ | vowel quality (/ɪ/→/i/) |
| Vietnamese | no final consonants → deletion | tonal, cluster reduction |

## Connections to Q162-Q170 Mock Design

**AND-frac hypothesis:** Whisper commits to acoustic evidence (AND-gate) for native phonemes.
For accented phonemes, the model falls back to language model priors (OR-gate), creating FAD bias.

**Mock design choices validated:**
- 6 L1 groups × 4 speakers = 24 speaker profiles → use as 6 group mock means + variance
- Expected AND-frac pattern: native ≈ 0.65, accented ≈ 0.52-0.57 (delta ≈ 0.08-0.13)
- Per-L1 WER from literature: Arabic ~23%, Mandarin ~18%, Hindi ~16%, Korean ~20%, Spanish ~14%, Vietnamese ~25%
- Pearson r(AND-frac, WER_gap) should be < -0.5 per Q162 DoD

**Mediation path (Q169):** Accent → low AND-frac → early t* (hallucination) → high WER
This mirrors the silence pathway: both cause the model to "give up" on acoustic commitment.

**Q167 (AFG metric):** AND-frac Fairness Gap = AND-frac(native) - AND-frac(accented)
Vietnamese and Arabic (highest WER) should show largest AFG.
