## What Was Built

`q175_conformal_and_frac.py` — Full conformal prediction calibration pipeline using AND-frac as nonconformity score. Implements:
1. CP calibration (split-conformal, α=0.10) on 100 cal / 100 test utterances
2. AUROC + F1 evaluation vs softmax baseline
3. Step-wise AND-frac word-level uncertainty analysis
4. Distribution shift evaluation (native vs accented)

## Novel Finding: Word-Level Uncertainty

Step-wise AND-frac profile comparison (8 bins):
- High-WER utterances: step profile ≈ 0.50 flat (model stuck in language-prior mode throughout)
- Low-WER utterances: step profile ≈ 0.79 flat (model grounded in audio throughout)
- **Late-step gap: +0.2917** (low−high)

**Interpretation:** Low-WER utterances maintain >50% higher AND-frac in every decode step. This is the step-wise resolution advantage: standard softmax CP can only flag utterances, not tokens. AND-frac enables word-level flagging (novel contribution for Paper A §4).

## Distribution Shift

AND-frac correctly flags high-WER utterances with 2x WER differential (flagged=0.501 vs unflagged=0.253 for native). Accented speech pattern similar. Softmax shows smaller differential for accented (flagged=0.380 vs unflagged=0.310), suggesting AND-frac is more robust to accent shift.
