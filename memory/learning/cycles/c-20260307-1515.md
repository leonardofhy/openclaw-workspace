## Problem
How can a cosmically diverse set of agents converge on shared pro tanto moral norms *without* assuming moral realism or shared culture?

## Method: Schelling Goodness Framework
- Extends Schelling focal-point theory from coordination games to morality
- **Key definition**: "X is Schelling-good" ≠ "X is good". It means: in a game where a diverse population tries to give *the same binary moral answer*, the plurality would land on "X is good"
- **Schelling participation effect**: as the participant pool grows, confidence in the focal answer increases recursively → more participation → stronger convergence
- **Cosmic Schelling version C(Q)**: ask what the cosmically general population (all plausible civilizations) would converge on when trying to agree
- **Scale-invariant adaptiveness** as the asymmetry filter: norms that help civilizations survive/grow across nested scales of organization tend to be Schelling norms

## Results / Main Claims
1. **Pro tanto framing**: all conclusions are ceteris paribus, not absolute. "Stealing is bad" ≠ "never steal", just that stealing is in the cons column
2. **Cosmic Schelling norms**: stealing→bad, Pareto-positive trade→good, killing→bad, honesty→good — all have short asymmetric arguments for why one side is more scale-invariantly adaptive
3. **Ties are rare**: 50.000% is almost never the answer unless the question is designed that way. Uncertainty about the answer ≠ tie
4. **Limited moral realism**: population-dependent but individual-invariant. Not mind-independent, but convergent for a fixed population
5. **AI alignment implications**: 
   - "Mutual respect between AIs and humans" → Schelling-good
   - "Irrevocably excluding a class of intelligent systems from coordination" → Schelling-bad
   - "Legibility, norm-discovery, bargaining over unilateral seizure" → Schelling-good
   - Framework suggests legible, interpretable AI is instrumentally convergent for cosmic coordination

## Connection to T5 / gc(k) Research

## Open Questions

2. Does the scale-invariance argument transfer to AI systems as sub-civilizational entities, or does it only apply at civilization scale?
3. Is the cosmic Schelling framework falsifiable in a scientific sense, or is it more like a normative framework?
4. Could gc(k) probe behavior itself be framed as a legibility signal — "this model listens here vs. guesses there" — to make it more Schelling-legible?
