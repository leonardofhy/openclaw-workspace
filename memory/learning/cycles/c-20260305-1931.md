## M7 — Spurious Correlation (`m7_spurious.py`)

**Design**: χ² test + articulatory coherence → harmonic mean = SpuriousScore(F)

Key components:
- `PHONEME_FEATURES`: 25-phoneme IPA articulatory feature matrix (16 binary dimensions)
- `articulatory_distance(p1, p2)`: 1 - feature_overlap / max_active (Choi et al. 2602.18899 inspired)
- `chi2_test(observed)`: normalized χ² against uniform phoneme distribution
- `MockSAE`: 3 archetypes: 50% specialized, 30% polysemantic, 20% noise
- `SpuriousScore = HarmonicMean(χ²_normalized, articulatory_coherence)`

**Mock results** (50 features, 2000 frames):
- Mean SpuriousScore = 0.204 (expected low — mock has 30% poly + 20% noise)
- 0 selective (≥0.6), 12 moderate (0.3–0.6), 38 spurious (<0.3)
- Interpretation: a good SAE should have most features in "selective" category

**Key design insight**: polysemantic features in mock are sampled from *different articulatory categories* by design — this properly tests the "spurious = acoustically unrelated" definition.

## M8 — Feature Geometry (`m8_geometry.py`)

**Design**: Phonological Cluster Geometry (intra/inter cosim ratio) + anti-podal detection + superposition index

Key components:
- `MANNER_GROUPS`: 5 articulatory classes (stop/nasal/fricative/approximant/vowel)
- `VOICING_PAIRS`: 5 canonical voiced/unvoiced pairs (/b-p/, /d-t/, /g-k/, /v-f/, /z-s/)
- `MockSAEWeights`: 3 archetypes: 60% phonological (manner-structured), 20% poly, 20% random
- `cluster_score = mean_intra_cosim / mean_inter_cosim` (target: > 2.0)
- `n_antipodal_pairs`: count pairs with cosim < -0.5

**Mock results** (100 features, d=64):
- Cluster Score: **13.559** ✅ (strong phonological structure)
- Mean intra-class cosim: 0.213 >> inter: 0.016
- Anti-podal pairs at -0.5: 0 ← **noted below**
- Superposition Index: 0.30 (moderate)

**Key insight re: anti-podal detection**:
At d_model=64, the voicing dimension is 1 out of 64 → cosine contribution ≈ 1/64 = 1.6% → voiced/unvoiced pairs won't reach -0.5 cosine at this dimensionality. Expected behavior for real Whisper SAEs (d=512+): voicing direction has more relative weight and anti-podal pairs SHOULD emerge. The -0.5 threshold is correct for Paper B but requires high-dimensional real SAEs to activate.

**Paper B framing**: "Audio features have stronger theoretical priors for geometry than text (voiced/unvoiced = binary, POA = categorical). AudioSAEBench M8 tests whether these priors are realized in trained SAEs — a novel testable prediction absent from SAEBench (text)."
