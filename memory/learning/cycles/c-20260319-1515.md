## Problem

Current SAE features for Whisper are not phoneme-pure. A feature may respond to /p/ but also weakly to /b/, /t/ (voicing/place bleed). Question: **How entangled are SAE features across phoneme categories?** Can RAVEL-style causal scrubbing quantify the bleed?

## Method (Conceptual Design)

**RAVEL for phoneme disentanglement:**
- RAVEL defines: `Cause` = variance explained by the target concept, `Isolate` = 1 - variance explained by distractor concepts  
- In original RAVEL: concepts are entity attributes (country, language, gender)
- Adaptation for phoneme-MDAS:
  - **Target concept**: phoneme identity (e.g., /p/ vs /b/ — voicing axis)
  - **Distractor concept**: place of articulation (/p/ vs /t/ — bilabial vs alveolar)
  - **Features**: SAE latent activations at layer L for Whisper
  - **Interventions**: swap activations from /p/ utterances → /b/ utterances at mean-ablated baseline

**Phoneme-MDAS dataset construction:**
1. Take minimal pairs: /p/–/b/, /t/–/d/, /k/–/g/ (voicing), /p/–/t/–/k/ (place)  
2. For each pair, extract SAE activations at every Whisper encoder layer  
3. Compute per-feature Cause(voicing) and Isolate(voicing | place) scores  
4. High Cause + High Isolate = **clean voicing detector** (disentangled)  
5. High Cause + Low Isolate = **bleed feature** (voicing + place entangled)

## Results (Expected)

- Early layers: low Cause, low Isolate (pre-phonemic features)
- Mid layers (4–6): peak Cause for voicing features; Isolate varies → reveals entanglement
- Late layers: reduced Cause (features become lexical, not phonemic)
- Hypothesis: t* collapse layers (gc(k) peak) correspond to layers where Isolate *drops* (features merge/compress)

## Connections

- **gc(k) and t***: T3's core metric (causal patching) should correlate with RAVEL's Cause score: high gc(k) ≈ high Cause for the patched phoneme  
- **AND-gate features (Q121/Q127)**: AND-gate features may be low-Isolate (respond to multiple phoneme dimensions simultaneously)  
- **FAD bias (Q123)**: Text-predictable phonemes = high Cause (easy to read from activations) but potentially low Isolate (bleed from surrounding context)  
- **Incrimination features (Q122)**: Top blame SAE features for misrecognition errors should show low-Isolate scores (entangled = confusable)
