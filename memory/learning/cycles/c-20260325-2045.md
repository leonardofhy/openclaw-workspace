# c-20260325-2045 | learn | Q175

## Task
Q175: gc(k) as Epistemic Uncertainty Estimator — Frame AND-frac as model-internal uncertainty signal. Compare to conformal prediction on WER calibration.

## Problem
AND-frac (fraction of "listening" attention heads active at decoding step k) already correlates with WER. But can it be framed as a **formal uncertainty estimator** with calibration guarantees? Conformal prediction (CP) offers a principled path.

## Method / Conceptual Synthesis

### Conformal Prediction Basics (for ASR)
- CP produces **prediction sets** with coverage guarantee: P(y ∈ Ĉ(x)) ≥ 1−α
- Requires only: (1) exchangeability/iid, (2) a **nonconformity score** s(x,y)
- For ASR: output is a transcript; uncertainty = probability the transcript is wrong
- Existing CP for NLP uses softmax confidence, length-normalized scores, etc.

### AND-frac as Nonconformity Score
Key insight: AND-frac at step k measures how many heads are "attending to audio" vs relying on language prior. Low AND-frac → model is **guessing** (language model mode). High AND-frac → model is **listening** (grounded in audio).

Proposed framing:
- s(x) = 1 − mean_k(AND-frac(k))  (utterance-level uncertainty)
- Higher s → more "guessing" → higher nonconformity
- Calibrate on held-out set: find threshold τ s.t. coverage ≥ 1−α
- Inference: flag utterances with s(x) > τ as "uncertain" (request human review)

### Comparison to Conformal WER Calibration
Standard CP for ASR (e.g., Angelopoulos 2021 framework):
- Score = 1 - softmax prob of top hypothesis
- OR: CER/WER on calibration set defines residual
- Produces confidence intervals on WER

AND-frac advantage:
1. **Mechanism-grounded** (not just softmax): captures WHERE the model fails (attention regime)
2. **Step-wise** resolution — can localize uncertainty to specific tokens/words
3. **No label needed at test time** — purely internal signal
4. **Complementary** — can be used as a feature alongside softmax in conformal framework

### Key Comparison Axes for Build
| Property | Softmax CP | AND-frac CP |
|----------|-----------|-------------|
| Distribution-free? | ✅ (by CP) | ✅ (by CP) |
| Needs output probs? | ✅ | ❌ (attention only) |
| Localizable? | ❌ word-level hard | ✅ step-wise |
| Interpretable? | Partial | ✅ (listen vs guess) |
| WER calibration? | Direct | Indirect (proxy) |

### Implementation Plan (for Build phase)
1. Extract AND-frac per utterance from existing gc(k) pipeline (already have eval harness)
2. Define s(x) = 1 - mean(AND-frac) across steps
3. Split dev set: calibration (50%) + test (50%)
4. Run CP: find τ at α=0.1 → coverage check
5. Compare to softmax-based nonconformity score
6. Measure: AUROC(AND-frac vs WER>0), calibration curve, coverage

### Open Questions
- Does AND-frac correlate with **word-level** WER or only utterance-level?
- Does the CP coverage hold under distribution shift (accented speech, noisy audio)?
- Can step-wise AND-frac produce **word-level confidence intervals** (novel contribution)?
- Is AND-frac better than token probability for detecting hallucinations specifically?
