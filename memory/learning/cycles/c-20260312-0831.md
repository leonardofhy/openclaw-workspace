## Theory: What is Causal Abstraction?

**Core (Geiger et al. NeurIPS 2021, arXiv:2106.02997):**
- Align neural representations with variables in an interpretable causal model
- Use **interchange interventions**: swap the value of neural repr from input A → B
- If the output changes as the causal model predicts → representation has causal structure
- IIA (Interchange Intervention Accuracy) = fraction of interventions where neural model matches causal model prediction

**Extended (Geiger et al. 2023, arXiv:2301.04709):**
- Generalizes from hard interventions → arbitrary mechanism transformations
- Unifies: activation patching, path patching, causal scrubbing, DAS, SAE
- Key insight: a **faithful abstraction** = low-level model consistently implements the high-level causal model across many interventions

## M9 Formal Definition (proposed)

**M9: Causal Abstraction Consistency at layer l**

Given:
- Causal model C with variables V (e.g., "audio content vector", "language prior weight")
- Neural model N with layers 1..L
- For each layer l: alignment function α_l: H_l → V (e.g., a linear probe)

**Consistency score:**
```
M9(l) = mean over interventions i of [N(do(α_l(h) = v_i)) matches C(do(V = v_i))]
       = IIA(layer l, causal model C)
```

**For adversarial detection:**
- Benign inputs: M9(listen_layer) should be HIGH (model follows audio causal model)
- Adversarial inputs: M9(listen_layer) drops (causal structure breaks — model stops faithfully using audio)
- Drop = causal inconsistency = early warning BEFORE harmful output

**Mock-computable version:**
- C = linear model: output = β × audio_vector + (1-β) × language_prior
- α_l = linear probe trained on benign data
- Intervention: replace h_l[audio] with h_l[noise] → check if output changes as predicted
- M9 = fraction of interventions where output change direction matches prediction

## Key Insight for Q083

**Why M9 lowers FPs:**
- M7 (SAE drift) catches "unusual feature activation"
- M7 alone fires on: accent shifts, background noise, code-switching (benign)
- M9 adds: "but is the model still causally using audio?"
- Adversarial: M7 ↑ AND M9 ↓ (model breaks away from audio grounding)
- Accented speech: M7 ↑ but M9 stays high (model still causally using audio)

**Prediction**: dual-factor FP rate < 0.5× single-factor FP rate at same recall
