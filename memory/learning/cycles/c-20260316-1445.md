## Paper: AudioSAE (arXiv:2602.05027, EACL 2026)

**Authors**: Aparin, Sadekova, Rukhovich, Yermekova, Kushnareva, Popov, Kuznetsov, Piontkovskaya

### Problem
SAEs used for interpretability in NLP; barely touched for audio. Do SAE features in audio models (Whisper, HuBERT) form stable, interpretable, causally useful representations?

### Method
- Train SAEs on **all encoder layers** of Whisper (ASR) and HuBERT (SSL)
- Measure: cross-seed stability, reconstruction quality, concept disentanglement
- Test: feature steering (causal intervention) on false speech detection

### Key Results
1. **>50% of features remain consistent across random seeds** — Schelling-style stability metric, measured per-layer
2. Reconstruction quality preserved despite sparsity
3. Concept erasure: removing only **19-27%** of features erases a target concept — sparse + modular
4. Feature steering: **70% reduction** in Whisper false speech detections with negligible WER increase
5. SAE features correlate with human **EEG activity** during speech perception
