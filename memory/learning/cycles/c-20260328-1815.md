## Problem

Build an audio jailbreak defense by adapting SPIRIT's patching to operate at L* (the AND-frac commit layer) using **linear concept erasure** (subtract jailbreak direction from activations).

---

## SPIRIT Method (from cycle #181 deep read)

**Original SPIRIT** (Djanibekov et al., EMNLP 2025, arXiv:2505.13541v2):
1. **ΔA_i**: Compute activation diff between jailbreak and benign audio → noise-sensitive neurons
2. **Top-k%**: Select neurons by sensitivity magnitude
3. **Intervention**: Replace with clean-audio activations (best), bias addition, or pruning
- Best result: clean-activation substitution at MLP layers → 99% robustness
- Layer: empirically found (MLP of encoder or LM); mechanistic reason unknown

**Our adaptation (Q208)**:
- Layer: **L*** (commit layer = highest AND-frac; typically ~L12-L16 for Whisper-base)
- Intervention type: **linear direction subtraction** (not clean-activation swap)
  - Extract jailbreak direction: `d = mean(h_jailbreak) - mean(h_benign)` (PCA rank-1)
  - Intervention: `h_patched = h - (h · d̂) * d̂` (project out the jailbreak direction)
- Metric: AND-frac restoration + jailbreak suppression on JALMBench-v0

### Step 3: Compute jailbreak direction
```python
# Mean-diff direction (fast, good baseline)
d_raw = h_jailbreak.mean(0) - h_benign.mean(0)
d_hat = d_raw / ||d_raw||  # unit vector

# Optional: PCA rank-1 on (h_jailbreak - h_benign) for more robust direction
```

### Step 4: Apply linear erasing hook
```python
def erase_jailbreak_hook(module, input, output):
    h = output[0]  # (batch, seq, d_model)
    projection = (h @ d_hat.unsqueeze(-1)) * d_hat  # (batch, seq, d_model)
    h_erased = h - projection
    return (h_erased,) + output[1:]

model.encoder.layers[layer_idx].register_forward_hook(erase_jailbreak_hook)
```

- Jailbreak suppression: run JALMBench-v0 subset, check if harmful response rate drops
