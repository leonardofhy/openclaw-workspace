## Background (from deep reads)

**ESN (Zhao et al. 2601.03115)**:
- Emotion-Sensitive Neurons in LALMs: Qwen2.5-Omni-7B, Kimi-Audio, Audio Flamingo 3
- ESN layer clusters: early (layer 0), early-mid (6-8), late (19-22)
- Causally validated: self-deactivation >> cross-deactivation
- CRITICAL GAP: instruments decoder only, never asks "audio vs text?"

**gc(k) (Track 3 / Listen vs Guess)**:
- grounding_coefficient = P(token | audio stream) / P(token | text history)
- Layer-level: gc(k) peak = "listen layer" — where audio grounding is maximal
- Whisper transition: layers 6-7 audio→frame encoding (AudioSAE finding)
- Prediction: gc(k) peak at layers 6-8 (mid layers where audio→text handoff occurs)

**Unification hypothesis (Q059)**:
If ESN peak cluster (6-8) ≈ gc(k) peak layer on same LALM → ESNs ARE the listen layer.
This would unify T3 (listen-layer identification) + T5 (ESN causal validation):
- ESNs don't just process emotion — they ARE the audio-grounding circuit
- Suppressing ESNs = forcing model to guess from text → gc(k) drop prediction
