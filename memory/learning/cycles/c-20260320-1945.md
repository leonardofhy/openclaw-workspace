## Problem
Does prosodic stress predict AND-gate fraction in Whisper's decoder?
- Stressed syllables = acoustically distinctive → model must listen → AND-gate
- Unstressed syllables = phonetically reduced → model uses LM prediction → OR-gate
This extends the FAD-bias framing to the prosodic dimension.

## Method (Design Spec)

### Data source
- **CMU Pronouncing Dictionary**: phoneme + ARPABET stress markers (0=none, 1=primary, 2=secondary)
- **LibriSpeech test-clean**: words mapped to CMU dict → phoneme-level stress labels
- **Whisper-base**: run inference, extract decoder cross-attention at layer k* (gc peak layer per Q001/Q002 results)

### Stratification
```python
# Group phonemes by stress level
groups = {
    'stressed':   phonemes with ARPABET stress == 1,
    'secondary':  phonemes with ARPABET stress == 2,
    'unstressed': phonemes with ARPABET stress == 0,
}
```

### AND-frac computation (already implemented in eval harness)
- AND-frac(t) = fraction of head activations above threshold at decoder step t
- Use gc peak layer k* from prior experiments (Q001: k*≈8 for Whisper-base encoder)
- Compute per-phoneme AND-frac → group-level mean ± SEM

### Statistical test
- Primary: Mann-Whitney U (stressed vs unstressed) — non-parametric, no Gaussian assumption
- Secondary: Pearson r(stress_level ∈ {0,1,2}, AND-frac) → expect positive correlation
- Effect size: Cohen's d (or rank-biserial r for U test)

### Mock validation (Tier 0, for this design)
```python
import numpy as np
from scipy import stats
np.random.seed(42)
# Simulate: stressed ~N(0.65, 0.08), unstressed ~N(0.52, 0.09)
stressed = np.random.normal(0.65, 0.08, 200)
unstressed = np.random.normal(0.52, 0.09, 300)
u_stat, p_val = stats.mannwhitneyu(stressed, unstressed, alternative='greater')
r_pb = 1 - 2*u_stat/(len(stressed)*len(unstressed))  # rank-biserial
# Expected: p < 0.001, r_pb > 0.3
```

### Success criterion
- AND-frac(stressed) > AND-frac(unstressed), Δ ≥ 0.08
- p < 0.05, rank-biserial r ≥ 0.20

## Results (Theory-Based)
**Mechanistic prediction:**
Stressed syllables carry contrastive acoustic information (f0 peak, duration lengthening,
spectral tilt) that cannot be recovered from LM context alone. Whisper's gc(k) framework
predicts such tokens require AND-gate activation because:
1. gc(k) peak measures the layer where audio signal causally dominates output
2. AND-gate = BOTH audio AND LM features jointly active → richer acoustic processing
3. Reduced/unstressed syllables are predictable from context → OR-gate suffices

**FAD-bias connection:**
High-stress phonemes = low FAD bias (model cannot predict from text distribution alone)
→ OR-gate fraction low, AND-gate fraction high.
Low-stress phonemes = high FAD bias (often schwa /ə/, which has near-uniform distribution
across phoneme classes in English) → OR-gate.
