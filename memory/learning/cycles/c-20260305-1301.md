## Research Question

Do **Emotion-Sensitive Neurons (ESNs)** in speech models simultaneously encode *multiple* acoustic attributes (emotion + speaker identity + phoneme content), making them poor at disentanglement — and does Audio-RAVEL's Isolate score capture this leakage?

**Motivation:** Zhao et al. (arXiv:2601.03115) identified "emotion-sensitive neurons" (ESNs) in speech LMs — neurons that fire strongly for emotional prosody. But emotional speech is confounded: angry speech is louder, faster, and often produced by speakers with particular demographic profiles. RAVEL's Cause/Isolate two-score framework (Huang et al., ACL 2024) was designed exactly to detect this "feature leakage" — where a feature that *causes* attribute A also *leaks* into attribute B.

**Hypothesis:** ESN SAE features will show:
- **Cause(F_ESN, emotion) ≈ 0.65–0.80** — ESNs do causally drive emotion prediction ✓
- **Isolate(F_ESN, emotion) ≈ 0.15–0.30** — ESNs fail to isolate emotion (leak into gender/speaker ID) ✗
- **Harmonic mean ≈ 0.30–0.45** — Well below the passing threshold of ~0.6 in text models (RAVEL paper baseline)
- **Implication:** Audio SAEs need *explicit* disentanglement training (Audio T-SAE style, Gap #17) to achieve acceptable isolation scores

---

## Why This Matters (AudioSAEBench Contribution)

This experiment defines **AudioSAEBench Category 0: Audio-RAVEL** — the most fundamental evaluation category:

1. **Cause alone = insufficient.** All existing audio SAE papers (AudioSAE, Mariotte, AR&D, Plantinga-PD) only measure Cause (via steering success). No paper measures Isolate.
2. **Isolation failure has safety implications.** If "emotion features" also encode speaker identity, then emotion-steering will cause unintended speaker identity drift → collateral damage in any safety/editing application.
3. **Fills the AudioSAEBench evaluation gap identified in Gap #22 and Gap #23.** The Cause × Isolate design from RAVEL gives us a quantitative standard that the field lacks.
4. **Connects to Paper A.** If ESN features have low Isolate scores, the gc(L) measurement in Paper A must account for potential emotion confounds in grounding attribution (confound check, §4.3).

## Attribute Set Design

### Target attribute: Emotion
- **ESN features** (from Zhao 2601.03115) = the feature set F under evaluation
- **Cause(F_ESN, emotion):** Does activating ESN features cause the model to output higher emotion classification probability?

### Leakage attributes (B ≠ emotion):
| Attribute | Why it leaks | Measure via |
|-----------|-------------|-------------|
| **Speaker gender** | Angry speakers in corpora ≈ more male; ESNs may encode gender direction | Probe accuracy before/after ESN patch |
| **Speaker identity** | Same speaker = consistent ESN activation pattern independent of emotion | Speaker ID classification change |
| **Phoneme content** | Emotional speech uses different phoneme distributions (f0-contour affects vowel duration) | ASR WER change / phoneme posterior shift |
| **Loudness (intensity)** | Anger = louder; ESN may encode loudness direction, not pure emotion | Energy extraction before/after patch |

**Key prediction:** Gender leakage will be highest (speaker-gender × emotion co-occurrence in all major emotion corpora — IEMOCAP, MSP-Podcast, EmoStar = all show M/F imbalance per emotion class).

## Stimuli Design

### Dataset: IEMOCAP (Interactive Emotional Dyadic Motion Capture)
- 12 hours; 5 sessions; 10 speakers (5M, 5F); 4 basic emotion classes (neutral, happy, sad, angry)
- Why: gold-standard for ESN research (Zhao et al. use IEMOCAP); perfect Cause/Isolate balance possible

### Stimulus pairing for interchange interventions:
```
Pair type A — emotion swap, same speaker:
  Source: Speaker F1, angry utterance → extract ESN activations
  Target: Speaker F1, neutral utterance → patch ESN features
  Expected Cause: anger posterior increases ✓
  Expected leakage: gender unchanged (same speaker) → Isolate(gender) will be high here

Pair type B — emotion swap, gender-crossing:
  Source: Speaker M1, angry utterance → extract ESN activations
  Target: Speaker F1, neutral utterance → patch ESN features
  Expected Cause: anger posterior increases
  Expected leakage: gender may shift toward M → Isolate(gender) will be LOW here
  ← This is the critical test case
```

**Condition matrix (2×2):**
| Same speaker | Different speaker |
|---|---|
| emotion swap A | emotion swap B (critical) |
| gender-constant | gender-crossing |

Minimum: 40 pairs per condition (160 total stimuli), balanced across emotion classes.

## Scoring Protocol (Audio-RAVEL)

Following RAVEL (Huang et al. 2024, ACL):

```python
# Cause(F_ESN, emotion):
# 1. Clean forward pass on Target (neutral)  → baseline emotion logit
# 2. Patch ESN features from Source (angry) → patched emotion logit
# 3. Ground truth: emotion class of Source = "angry"
cause_score = (patched_emotion_correct - baseline_emotion_correct) / (1 - baseline_emotion_correct)

# Isolate(F_ESN, emotion):
# For each leakage attribute B in {gender, speaker_id, phoneme, loudness}:
# 1. Clean forward pass on Target → baseline attribute B prediction
# 2. Patch ESN features from Source → patched attribute B prediction
# 3. Measure: does B change?
isolate_B = 1 - |predict_B(patched) - predict_B(clean)|  # lower change = higher isolation
isolate_score = mean(isolate_B, over all leakage attributes B)

# Final Audio-RAVEL score:
audio_ravel_score = 2 * cause_score * isolate_score / (cause_score + isolate_score)  # harmonic mean
```

**Baseline ceiling:** Apply MDAS (Multi-task DAS from RAVEL) to Whisper residual stream — simultaneously learn orthogonal subspaces for all attributes. Expected MDAS Audio-RAVEL score: ~0.60–0.70 (upper bound for non-SAE method).

## Paper A §4.3 Implication

If ESN features have Isolate(gender) < 0.30:
- A model doing "audio grounding" via emotion-sensitive neurons may show high gc(k) simply because angry speakers have distinctive acoustic profiles, not because the model understands semantic content
- **Confound control:** Paper A must check that gc(k) is not driven by prosodic/energy features confounded with emotion
- **Mitigation:** Filter ESC-50 stimuli to avoid emotionally-loaded sounds (dog barking vs calm speech); use NEUTRAL prosody whenever possible
