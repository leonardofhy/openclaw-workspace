## Paper: ESN (Zhao et al., arXiv 2601.03115)

**Problem:** Do LALMs encode emotion via dedicated neurons, and is this encoding causally verifiable?

**Method:**
- 4 neuron selectors: frequency (high activation count), entropy (low entropy = selective), magnitude (top-k activation), contrast (emotion A vs all others)
- Intervention types: ablation (zero) → degradation of target emotion; gain amplification → steer toward emotion
- Tested on Qwen2.5-Omni, Kimi-Audio, Audio Flamingo 3

**Results:**
- ESNs exist causally: ablating emotion-X degrades emotion-X > other emotions
- Gain amplification confirms directionality
- Non-uniform layer clustering: ESNs not evenly distributed across layers
- Partial cross-dataset transfer: some ESNs generalize across datasets

**Key gap for Q040:** ESNs are identified within emotion space but NOT tested for speaker-attribute leakage. High pitch = female = happy — emotion ESNs may actually encode speaker gender as a confound. This is the RAVEL Isolate failure we're trying to quantify.

## Q040 Design: Audio-RAVEL × ESN Leakage Scorecard

**Hypothesis:** ESNs selected by contrast-based selector have high Cause (emotion) but low Isolate (gender leaks), due to speaker-emotion co-occurrence in standard datasets (e.g., MELD, IEMOCAP).

### 2×2 Control Matrix

```
               Speaker: Female    Speaker: Male
Emotion: Happy     [FF_H]           [FM_H]
Emotion: Angry     [FF_A]           [FM_A]
```

**Patching target:** ESNs selected for "Happy" (top-k by contrast selector)

### RAVEL Metrics

| Metric | Formula | Expected (leakage hypothesis) |
|--------|---------|-------------------------------|
| **Cause** | P(emotion flips happy→neutral \| ESN ablated) | High (~0.7) |
| **Isolate** | 1 - P(gender prediction changes \| ESN ablated) | Low (~0.4) — gender also affected |
| **Net score** | Cause + Isolate | ~1.1 (below RAVEL threshold of 1.4) |

### Experimental Protocol (Tier 0 — design only, no GPU)

1. **Dataset construction:** Sample MELD/IEMOCAP 4-cell balanced: {happy,angry} × {male,female}; ~50 samples per cell (200 total)
2. **ESN identification:** Use contrast selector from ESN paper (available in their released code or reimplementable); identify top-k ESNs for "happy" label
3. **Causal patching:**
   - Ablation: zero-out ESNs for "happy" → measure Δ emotion accuracy
   - Gender probe: linear probe trained on speaker-gender labels → measure Δ gender accuracy after same ablation
4. **Score computation:** Cause = ΔEmotion / baseline_emotion; Isolate = 1 - (ΔGender / baseline_gender)
5. **Controls:** shuffle-ablation baseline (random k neurons) to verify specificity

### Expected Leakage Pattern

If ESNs leak into gender:
- Cause ≈ 0.7 (ablation degrades happy recognition)
- Isolate ≈ 0.3 (ablation also changes gender probe → emotion feature encodes gender)
- Pattern: high-pitch "happy" samples drive ESN identification, low-pitch "happy" samples are false negatives
- Implication for AudioSAEBench: Category 0 score reflects emotion-speaker entanglement, not pure emotion encoding

### Connection to Tracks

- **T5 (Listen-Layer Audit):** If ESNs are entangled with speaker attributes, "emotion control via ESN" is unreliable → safety concern (affective manipulation probe degrades unexpectedly on atypical speakers)
- **T3 (Listen vs Guess):** gc(k) may be confounded by speaker-emotion co-occurrence; need gender-balanced test sets
- **AudioSAEBench contribution:** This scorecard → Category 0 (Cause + Isolate under controlled attribute conditions)

### Open Questions

1. Does the contrast selector produce *less* leakage than frequency/entropy selectors?
2. Is leakage consistent across Qwen2.5-Omni, Kimi-Audio, Audio Flamingo 3?
3. Can we find emotion ESNs with Isolate > 0.6 by conditioning on speaker-balanced data?
