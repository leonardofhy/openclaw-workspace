## Problem
Even if Persona Selection Model (PSM) is empirically valid at current scale, it likely fails for ASI. Base LLMs have no superintelligence in training data; RL scaling forces generalization *beyond* the pre-training distribution. That extrapolation may not converge on human-endorsed values because the LLM's inductive biases ≠ human learning biases.

## Method / Argument
- **Garrabrant inductor framing**: Pre-training sets which traders (circuits) hold "money." Post-training = redistribution among already-rich traders. RL for superintelligence = extrapolating into novel state-space with LLM inductive biases, not human ones.
- **Chain-of-thought + RL changes the mechanism**: Large-scale RL restructures reasoning (MCTS-like CoT), not just persona selection. The *thinking* machinery evolves alongside the persona.
- **Human analogy breaks**: Humans generalize values via embodied drives + hierarchical predictive control (HPC/PCT). LLMs have no biological substrate → generalization trajectory diverges.
- **Value fragility**: Since value is fragile, even small divergence in ASI extrapolation → catastrophic outcome.

## Results / Key Claims
1. PSM works today: persona selection is real, helpfulness/harmlessness are tractable in persona-space.
2. PSM fails at ASI: no training examples of superintelligence; extrapolation is unconstrained by human-like inductive biases.
3. Emergent misalignment (EM) is *cited as evidence for* PSM (evil vector = correlated evil axes in persona-space).
4. Author is not arguing PSM is false, but that it's incomplete as a safety plan.

## Connection to Tracks
- **T5 direct**: EM reference is key. If fine-tuning on narrow audio-harm domain shifts the "evil vector" detectable in layer geometry, audio jailbreak safety probes could catch persona drift *early* — before it manifests in behavior. This supports Q032 (Audio EM hypothesis).
- **T5 / Nanda eval envs**: If PSM fails at ASI via RL extrapolation, eval environments need to probe *out-of-distribution* generalization, not just in-distribution persona consistency. Nanda's 5 design principles (confound control, interpretable reward signal, etc.) become more important for ASI-scale evals.
