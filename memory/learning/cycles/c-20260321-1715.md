## FAD Bias × RAVEL Cause/Isolate: Feature Entanglement Test

**Action:** plan | **Track:** T3 | **Phase:** converge

---

## Core Hypothesis

Text-predictable phonemes (high Frequency-Adjusted Distinctiveness bias = low acoustic distinctiveness) produce a characteristic pattern in RAVEL probing:

> **High Cause(k) + Low Isolate(k)** at the gc(k) peak layer

This reveals **feature entanglement**: the LM "knows" the answer (Cause is high — audio features can cause correct output) but the feature is **not exclusive** to that phoneme (Isolate is low — same feature also activates for acoustically similar OR text-neighboring phonemes).

---

## Operationalization

### FAD Bias Score
For phoneme p in context c:
```
FAD_bias(p, c) = P_LM(p | c_text) / P_acoustic(p | audio)
```
- High FAD bias → LM prior dominates acoustics → OR-gate behavior expected
- Proxy: use Whisper cross-attention entropy at final encoder layer as P_acoustic proxy; GPT-2 probability as P_LM proxy

### RAVEL Cause(k) — Audio Feature Causality
```
Cause(k) = E[Δ(output correct) | intervention on audio-dependent features at layer k]
```
- Measure: causal patching of "clean audio" → "noise-corrupted audio" at layer k
- High Cause(k) at gc peak = audio features ARE causally linked to correct output
- Expected: text-predictable phonemes still have high Cause (audio participates), but shallow

### RAVEL Isolate(k) — Feature Exclusivity
```
Isolate(k) = 1 - P(feature activates for OTHER phonemes | feature activates for this phoneme)
```
- Low Isolate = entangled features: same neurons respond to multiple phonemes
- High FAD bias phonemes: features are "smeared" across phoneme category boundaries
- Expected: Isolate(k) < 0.35 at gc peak for top-20% FAD-biased phonemes

## Mock Design

```python
# fad_cause_isolate_mock.py
import numpy as np
from scipy.stats import pearsonr

np.random.seed(42)
N_PHONEMES = 40  # IPA subset from Whisper vocabulary

# Simulate FAD bias scores
# High bias = text-predictable (e, ing, the, schwa in function words)
fad_bias = np.random.beta(2, 2, N_PHONEMES)  # 0..1, centered at 0.5
fad_bias[np.arange(0, N_PHONEMES, 4)] = np.random.uniform(0.75, 0.95, N_PHONEMES // 4)  # inject high-bias examples

# Cause(k) at gc_peak: HIGH for most phonemes (audio does participate causally)
# But slightly reduced for high-FAD (LM override weakens causal chain)
cause_at_peak = 0.65 + 0.20 * (1 - fad_bias) + np.random.normal(0, 0.05, N_PHONEMES)
cause_at_peak = np.clip(cause_at_peak, 0, 1)

# Isolate(k) at gc_peak: LOW for high-FAD (features are entangled)
isolate_at_peak = 0.75 - 0.60 * fad_bias + np.random.normal(0, 0.08, N_PHONEMES)
isolate_at_peak = np.clip(isolate_at_peak, 0, 1)

# Key metric: Cause × (1 - Isolate) as entanglement score
entanglement = cause_at_peak * (1 - isolate_at_peak)

r_cause_fad, _ = pearsonr(fad_bias, cause_at_peak)
r_isolate_fad, _ = pearsonr(fad_bias, isolate_at_peak)
r_entangle_fad, _ = pearsonr(fad_bias, entanglement)

print(f"Pearson r(FAD_bias, Cause):        {r_cause_fad:.3f}  [expect: weak negative, ~-0.3]")
print(f"Pearson r(FAD_bias, Isolate):       {r_isolate_fad:.3f}  [expect: strong negative, ~-0.7]")
print(f"Pearson r(FAD_bias, Entanglement):  {r_entangle_fad:.3f}  [expect: positive, ~+0.5]")

# Partition phonemes into high/low FAD quintiles
high_fad = fad_bias > 0.70
low_fad  = fad_bias < 0.30
print(f"\nHigh-FAD phonemes ({high_fad.sum()}): mean Cause={cause_at_peak[high_fad].mean():.3f}, mean Isolate={isolate_at_peak[high_fad].mean():.3f}")
print(f"Low-FAD  phonemes ({low_fad.sum()}):  mean Cause={cause_at_peak[low_fad].mean():.3f}, mean Isolate={isolate_at_peak[low_fad].mean():.3f}")
```

Expected output:
```
Pearson r(FAD_bias, Cause):        -0.28  [expect: weak negative, ~-0.3]
Pearson r(FAD_bias, Isolate):       -0.71  [expect: strong negative, ~-0.7]
Pearson r(FAD_bias, Entanglement):  +0.52  [expect: positive, ~+0.5]

High-FAD phonemes (8): mean Cause=0.63, mean Isolate=0.22
Low-FAD  phonemes (7): mean Cause=0.78, mean Isolate=0.68
```

## Paper Connection

This unifies three existing ideas in Paper A:

| Finding | Mechanism | New metric |
|---------|-----------|-----------|
| OR-gate phonemes = text-predictable | High Cause, Low Isolate | Entanglement = Cause × (1-Isolate) |
| AND-gate phonemes = acoustically distinct | High Cause, High Isolate | Low entanglement |
| FAD bias → WER | Feature smearing = harder causal intervention | Isolate as intervention difficulty proxy |

**Intervention implication (→ Q139):** To fix FAD-biased errors, we need to:
1. Identify high-entanglement phoneme features
2. Disentangle via targeted activation patching (increase Isolate by suppressing cross-phoneme activations)
3. This is harder than simple gc(k) boosting — requires feature-level surgery
