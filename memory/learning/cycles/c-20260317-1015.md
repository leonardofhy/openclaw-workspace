## Background & Motivation

**Paper A Claim 2** (working): At the gc(k) peak layer, audio-LLM features exhibit
a mechanistic "listen-or-guess" split: AND-gate features (require audio AND text context)
dominate clean decoding, while OR-gate features (either audio OR text suffices) dominate
cascade/failure modes.

**Cascade Equivalence Hypothesis**: A "cascade" (model ignores audio, guess from text alone)
is mechanistically equivalent to OR-gate dominance at the gc peak. If audio channel is
attenuated early (t* collapse, backdoor, jailbreak), the model falls back to OR-gate routing.

**Bridge**: Define `cascade_degree(k) = 1 - AND_gate_fraction(k)` at layer k=gc_peak.
- AND_gate_fraction = |{f: f is AND-gate at gc peak}| / |{f: all features at gc peak}|
- High AND fraction → high audio dependence → clean listen → cascade_degree ≈ 0
- Low AND fraction → OR dominance → text-only fallback → cascade_degree ≈ 1

Prediction: **cascade_degree is negatively correlated with gc(k) at peak layer**.

### Setup
- MicroGPT toy model (existing scripts from Q080/Q082/Q092)
- 5 seeds × 2 conditions: clean audio, degraded audio (SNR=0dB)

### Step 1: Measure AND-gate fraction per condition
```python
# mock_cascade_degree.py
from microgpt_sim import build_mock_features, gc_at_layer

seeds = [42, 123, 456, 789, 1337]
results = []
for seed in seeds:
    for condition in ['clean', 'degraded']:
        feats = build_mock_features(seed, condition)
        gc_peak_layer = gc_at_layer(feats).argmax()
        # AND-gate: feature activates only when audio AND text both present
        # OR-gate: feature activates when audio OR text present
        and_frac = compute_and_gate_fraction(feats, gc_peak_layer)
        cascade_deg = 1 - and_frac
        gc_val = gc_at_layer(feats)[gc_peak_layer]
        results.append({
            'seed': seed, 'condition': condition,
            'gc_peak': float(gc_val),
            'and_fraction': float(and_frac),
            'cascade_degree': float(cascade_deg)
        })
```

### Step 2: Correlation test
```python
import numpy as np
from scipy.stats import pearsonr

cascade_scores = [r['cascade_degree'] for r in results]
gc_peaks = [r['gc_peak'] for r in results]
r, p = pearsonr(cascade_scores, gc_peaks)
# Hypothesis: r < -0.8 (strong negative), p < 0.05
```

### Step 3: Within-seed delta
```python
# Compare clean vs degraded within same seed
for seed in seeds:
    clean = next(r for r in results if r['seed']==seed and r['condition']=='clean')
    deg   = next(r for r in results if r['seed']==seed and r['condition']=='degraded')
    delta_cascade = deg['cascade_degree'] - clean['cascade_degree']
    delta_gc      = deg['gc_peak'] - clean['gc_peak']
    # Expect: delta_cascade > 0, delta_gc < 0 (both degrade together)
```
