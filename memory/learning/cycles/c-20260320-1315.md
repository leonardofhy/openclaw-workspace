## Problem

The ENV taxonomy (from Q106) labels persistent offender SAE features as:
- ENV-1: high-influence "hub" features — many downstream dependents  
- ENV-2: medium influence  
- ENV-3: isolated / low-influence features — few connections

GSAE (Graph-SAE) represents SAE feature interactions as a directed graph where edges encode causal influence. The question: **do ENV labels map onto GSAE cluster roles?**

---

## Design: ENV ↔ GSAE Role Mapping

### Hypothesis

| ENV Type | GSAE Cluster Role | Structural Signature | Intervention |
|----------|------------------|----------------------|--------------|
| ENV-1 | **Hub node** | High out-degree, central betweenness, bridges clusters | Reduce hub edges (targeted ablation) |
| ENV-2 | **Bridge node** | Medium out-degree, connects two dense regions | Prune selectively |
| ENV-3 | **Isolated cluster** | Low degree, weak incoming + outgoing, peripheral | Safe to prune entirely |

### Mechanistic Rationale

**ENV-3 = isolated cluster:**
- These features activate persistently but cause little downstream damage
- In GSAE terms: low betweenness centrality, few edges → they're isolated
- Pruning them is safe — minimal collateral
- Matches jailbreak-pruning observation (Q128): ENV-3 pruning restores t*

**ENV-1 = hub node:**
- These features are the "conductors" — activation spreads to many others
- In GSAE: high out-degree, large betweenness → intervention must be surgical
- Cannot prune — must reduce edge weights (activation steering, not ablation)
- Strategy: dampen, not delete

**ENV-2 = bridge node:**
- Intermediate case
- Connects subgraphs — removal may fragment useful pathways
- Requires case-by-case evaluation

---

## Intervention Protocols by ENV Type

### ENV-3 (Isolated): Full Prune
```
if gsae_degree(f) < threshold_low:
    prune(f)  # zero out activation
```
Expected: t* recovery (Q128), minimal WER impact

### ENV-1 (Hub): Edge Reduction
```
if gsae_out_degree(f) > threshold_high:
    scale_activation(f, factor=0.3)  # don't zero, just dampen
```
Expected: reduces cascade spread while preserving phoneme decoding

### ENV-2 (Bridge): Selective Pruning
```
if threshold_low < gsae_degree(f) < threshold_high:
    evaluate_subgraph_impact(f)
    if impact_score < 0.2: prune(f)
    else: dampen(f, factor=0.5)
```
