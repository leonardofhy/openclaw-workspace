## Method (Preacher & Hayes 2008 bootstrap mediation)

**Variables:**
- X = accent (0=native, 1=L2)
- M = AND-frac at L* (mediator)
- Y = WER (outcome)

**Three paths:**
- Path a: X → M (OLS: does accent lower AND-frac?)
- Path b: M → Y controlling X (OLS multi: does AND-frac → lower WER?)
- Path c': X → Y direct (accent effect after controlling M)
- Total c = c' + a×b

**Indirect effect** = a×b (the mediated part)
**Proportion mediated** = a×b / c → need ≥30% for Q194

**Bootstrap CI:**
- N=1000 resamples with replacement
- Compute a×b each resample → percentile CI
- Significant if CI excludes zero
