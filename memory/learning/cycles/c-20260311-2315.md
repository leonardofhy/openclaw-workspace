# c-20260311-2315 — Plan: Q044 Microgpt × CoT × gc(k) Spec

**Action**: plan | **Task**: Q044 | **Track**: T3
**Research question**: Does CoT prefix suppress the gc(k) phonological grounding signal in microgpt? At what layer? Via which circuit?

## Background

In 7B models (Whisper/Qwen2-Audio) we observe gc(k) signal correlates with phonological grounding — but the residual stream is opaque. Microgpt (≤1M params, fully inspectable) gives circuit-level resolution. CoT chains ("Let me think step by step…") may route attention away from acoustic features before the grounding layer fires.

## Protocol

### Model Setup
- **microgpt**: 4-layer transformer, ~500K params, trained on toy phoneme-token pairs
- **Vocabulary**: 32 phoneme tokens + 8 CoT tokens ("think", "step", "reason", "therefore", etc.)
- **gc(k) hook**: patch into each layer's residual stream (same hook as `whisper_hook_demo.py` logic)

### Conditions (2×2 + control)
| Condition | Input | CoT Prefix |
|-----------|-------|-----------|
| C1 (baseline) | audio token seq | none |
| C2 (CoT short) | audio token seq | "think:" (1 token) |
| C3 (CoT long) | audio token seq | "think step by step:" (4 tokens) |
| C4 (CoT only) | CoT tokens only | full |
| C5 (null — no audio) | random tokens | none |

### Measurements
1. **gc(k) per layer** — record grounding coefficient at each of 4 layers for C1–C5
2. **Attention headmap** — for C1 vs C3: which heads attend to CoT vs audio tokens at each layer?
3. **Feature ablation** — zero-ablate CoT-attending heads in isolation; measure gc(k) recovery

### N=5 / F=4 Scoring
- N (novelty=5): Circuit-level CoT-suppression mechanism completely unknown at scale → high novelty
- F (feasibility=4): Microgpt + CPU hook = Tier 1 runtime, <3 min; only blocker is scaffold code

### Expected Results
- gc(k) drops at intermediate layer when CoT prefix is long (C3 < C1 < C4~C5)
- Specific heads (cross-attention on CoT tokens) responsible — ablating them restores gc(k)
- Provides causal story: "CoT suppresses grounding at layer L via head H_i → eroded phonological signal → downstream transcription drift"
