# c-20260323-1245 — Q155: AND-frac Scaling with Whisper Model Size

**Task**: Design doc + mock: AND-frac at gc_peak across Whisper base→small→medium→large
**Track**: T3 | **Phase**: converge | **Action**: build (Tier 0)

## Hypothesis

Larger Whisper models commit to audio evidence earlier and more reliably.
Mechanistically: larger models have more features per layer → more redundancy → AND-gate condition
(both audio-grounded AND linguistically plausible) is easier to satisfy simultaneously.

**Prediction**: AND-frac at gc_peak scales monotonically with model size.

### AND-frac at gc_peak — Mock Design

AND-frac(model_size) ≈ f(log(params), layer_depth, head_count)

Empirical scaling law candidate:
  AND-frac(k*) ≈ A × log10(params_M) + B

Where:
- k* = gc_peak layer index (normalized to layer depth)
- AND-frac = fraction of top-K SAE features that show AND-gate behaviour
  (causal from audio AND correlation with text label)

### Mock Simulation

See `and_frac_scaling_mock.py` in this cycle folder.

Expected trend (mock numbers, to be validated with real runs):
- base (74M):     AND-frac ≈ 0.31
- small (244M):   AND-frac ≈ 0.38
- medium (769M):  AND-frac ≈ 0.44

Slope ≈ +0.062 per decade of parameters.

### Confounds to Control

1. **Layer normalization depth**: gc_peak shifts to later layers in larger models
   → normalize k* = gc_peak_layer / total_encoder_layers
2. **WER baseline**: larger models have better WER → AND-frac may co-vary with accuracy
   → hold WER constant by testing on medium-difficulty subset (30-50 WER range)
3. **SAE quality**: SAEs for small/medium/large don't exist yet
   → mock: use random projection to simulate SAE feature space

### Connection to Paper A

This directly supports the "gc(k) as universal mechanism" claim:
- If AND-frac scales with model size, then **larger models listen harder** (more audio-grounded)
- This predicts: scale alone does not fix hallucination (still need AND-gate at right layers)
- Policy implication: AND-frac monitoring = cheap proxy for hallucination risk across model sizes
