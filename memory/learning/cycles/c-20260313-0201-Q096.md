## Core Claim (Q096)

**Hypothesis:** At the gc(k) peak layer (the cross-modal integration point), AND-gate SAE features
disproportionately represent audio-dependent phonemes, while OR-gate features represent
text-predictable phonemes.

**Mechanistic chain:**
1. FAD bias (Gui et al. 2602.23958): Whisper's encoder is biased toward text-predictable patterns.
   It becomes "acoustically blind" to features that can be inferred from language context.
2. At the gc(k) peak: this is where audio information is maximally integrated causally (forced-
   alignment measurement with gc = P(correct | audio) - P(correct | text-only baseline)).
3. AND-gate = audio-essential feature: fires only when BOTH audio AND context are present.
   OR-gate = context-sufficient feature: fires even when audio is degraded (context alone).
4. **Prediction:** OR-gate fraction inversely correlates with text-predictability of the phoneme.
   Text-predictable phonemes (e.g., "the" → /ðə/) → OR-gate (language prior sufficient).
   Audio-dependent phonemes (homophones, rare phones, cross-lingual) → AND-gate.

---

## Mathematical Formulation

Let P_text(p) = text-predictability of phoneme p (e.g., unigram/bigram LM probability given context).
Let fAND(p, k) = fraction of AND-gate SAE features that encode phoneme p at layer k.
Let fOR(p, k) = fraction of OR-gate SAE features that encode phoneme p at layer k.

**Main test:**
  Pearson r between P_text(p) and fOR(p, k*) where k* = gc peak layer.
  Prediction: r > 0 (high text-predictability → high OR-gate fraction).
  Prediction: r < 0 for AND-gate fraction.

**FAD invariance connection:**
  The "invariance set" of the encoder (Gui et al.) = set of perturbations to which gc ≈ 0.
  These should map to high OR-gate fraction at k*: model can "fill in" from context.
  Phonemes in the invariance set are NOT measured by FAD → explains FAD's perceptual misalignment.

- Passthrough: active in clean, drops in noisy-audio, does NOT recover

## Paper A §4 Addition (FAD Bias Connection)

**Section title:** §4.3 — Mechanistic Basis of Encoder Bias

**Argument:**
> FAD encoder bias (Gui et al., 2026) can be operationalized mechanistically:
> the "invariance set" of Whisper's encoder corresponds to phonemes whose SAE features
> at the gc(k) peak are predominantly OR-gated. These phonemes are acoustically accessible
> via language priors alone, making them invisible to perceptual quality metrics.
> AND-gate features at k* are the mechanistic substrate of acoustic sensitivity.
> This provides the first circuit-level explanation for why FAD with Whisper fails on
> certain perceptual dimensions — the relevant phonemes have no AND-gate representation.

**Cite:** Gui et al. 2602.23958 (FAD encoder bias)
**Connects to:** Q080 (AND-gate Schelling stability), Q085 (collapse onset), Q089 (AND vs OR temporal onset)
