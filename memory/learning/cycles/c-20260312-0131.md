## Paper: The Cascade Equivalence Hypothesis
**"When Do Speech LLMs Behave Like ASR→LLM Pipelines?"**  
Jayadev Billa | arXiv:2602.17598 | Interspeech 2026 submission | cs.CL + eess.AS  
Submitted 2026-02-19, updated 2026-03-05 (10 pages)

---

## Problem
Speech LLMs (Qwen2-Audio, Ultravox, Phi-4-Multimodal, Gemini) promise "better than cascades" because raw audio has prosody/emotion/emphasis beyond transcript. Does this hold, or do they converge on implicit text representations anyway?

**Prior confound**: comparing speech LLM vs. Whisper+GPT mixes backbone differences with architecture differences. Nobody controlled for this.

---

## Method
### Key contribution 1: Matched-backbone testing
- Pair Whisper with the SAME LLM backbone used inside the speech LLM
  - Qwen2-7B for Qwen2-Audio
  - Llama-3.1-8B for Ultravox
  - Phi-4-mini for Phi-4-Multimodal
- This isolates architecture (connector + encoder) from backbone reasoning

### Cascade Equivalence Hypothesis (CEH)
Formal: when I(A;Y|T) ≈ 0 (text-sufficient tasks), the speech LLM SHOULD behave identically to its matched cascade.  
- Acoustic surplus: ΔI_Y = I(A;Y) - I(T;Y)
- Text-sufficient: ΔI_Y ≈ 0 (factual QA, topic classification, sentiment)
- Text-insufficient: ΔI_Y > 0 (emotion recognition, sarcasm)

### Evaluation
- 6 tasks: TriviaQA, AG News, SST-2, CommonsenseQA (text-sufficient); MELD, MUStARD (text-insufficient)
- 6,300 samples, 27,300 audio files
- Noise: multi-talker babble at 0, 5, 10, 15 dB SNR
- Metrics: Cohen's κ (per-example agreement), conditional error overlap P(same wrong | both wrong), McNemar's test

### Mechanistic analysis
- **Logit lens**: hidden states at 9 layers → unembedding matrix → do top-k tokens match transcript words?
- **Probes**: acoustic (energy, pitch via R²), CTC probe (text decodability = 1-CER), BoC (bag-of-chars R²)
- **LEACE erasure** (LEAst-squares Concept Erasure): surgically remove text-predictive subspace → does performance collapse? If yes → text representations are causally necessary (not epiphenomenal)
  - BoC erasure, proxy erasure, CTC erasure
  - Control conditions: random erasure, acoustic erasure

---

## Results (from abstract + intro)
1. **Backbone confound inflates divergence by up to +0.13κ** — much of apparent speech LLM advantage disappears when using matched backbone
2. **Cascade equivalence is a spectrum across architectures** — CEH mostly holds on text-sufficient tasks
3. **Text representations are causally necessary** — LEACE erasure destroys performance; logit lens shows transcript emerging in hidden states
4. **Under noise: cascades WIN** — clean-condition advantage reverses by up to **7.6% at 0dB SNR** — speech LLMs are more noise-sensitive than Whisper-based cascades
