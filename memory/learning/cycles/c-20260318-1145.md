## Paper: "Discovering and Causally Validating Emotion-Sensitive Neurons in Large Audio-Language Models"
**Authors**: Zhao, Schuller, Sisman | arXiv:2601.03115 | Jan 2026 | 16pp

---

## Problem
Lack mechanistic account of how LALMs encode emotion. Previous work: probing (correlational). This work: neuron-level causal validation.

## Method
- Models: Qwen2.5-Omni, Kimi-Audio, Audio Flamingo 3
- Selector comparison: frequency, entropy, magnitude, **contrast-based** (best)
- Intervention: (1) ablation → measure per-class degradation; (2) gain amplification → steer predictions
- Multi-benchmark evaluation (IEMOCAP, MER2023, etc.)

## Results
1. **Causal isolation confirmed**: ablating emotion-e neurons degrades class-e disproportionately, preserves others
2. **Gain amplification** steers predictions to target emotion → ESNs are *causal handles*
3. ESNs show **non-uniform layer-wise clustering** (mid-to-late layers, not uniform)
4. **Partial cross-dataset transfer** → ESNs encode general emotion signal, not just dataset artifacts
5. Effects scale systematically with intervention strength

## Connection to Tracks

### Q118 (Emotion neurons × AND/OR gates)
- This paper finds causal ESNs in LALMs → we can now ask: are these AND-gates (require audio+context) or OR-gates (audio alone triggers)?
- The **contrast-based selector** is the best → maps to AND-gate identification: AND-gate features fire only when BOTH audio signal AND linguistic context align
- **Hypothesis refinement**: ESNs selected by contrast method (not frequency) are more likely AND-gates — because contrast implicitly penalizes features that fire independently of modality
- Cross-dataset transfer of ESNs supports robustness → AND-gate features tend to be more transferable (less task-specific)
- **Mock experiment extension**: emotion_gate_mock.py should also implement contrast selector comparison vs frequency selector → test if contrast-selected features have higher AND-fraction
