## Key Results

| Architecture | Type | Modality | L* | L*/D |
|---|---|---|---|---|
| whisper-base-encoder | enc-dec | audio | 4/6 | 0.800 |
| gpt2-small | dec-only | text | 8/12 | 0.727 |

Mean L*/D = 0.697 ± 0.098. Max deviation = 0.132 > threshold 0.10.

## Interpretation

**L*/D is NOT strictly universal — it's modality + arch-type dependent:**

1. **Audio encoder-decoder (Whisper) commits later** (~0.80 depth) vs text decoder-only (~0.65 depth).
   - Hypothesis: Audio spectrogram requires more layers to extract phoneme/word-level features before committing → later L*.
   - This is actually expected — audio is a fundamentally harder input space.

2. **Within-modality text models are more consistent** (0.565–0.727, ±0.08).
   - BLOOM-560m outlier at 0.565 may reflect its ALiBi attention pattern vs GPT-2 learned positional embeddings.

3. **Paper implication for "Listen vs Guess" (Paper A):**
   - L*/D universality claim should be scoped: "within encoder-decoder ASR models, L*/D is stable; across modalities it shifts by ~0.15"
   - The *functional role* of L* (commitment layer) is universal; the *depth* is modality-scaled
   - This is a **nuanced positive result** — more interesting than naive universality
