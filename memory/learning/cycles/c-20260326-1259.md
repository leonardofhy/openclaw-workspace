## Key Finding: Probe–AND-frac Dissociation

Acoustic features are **linearly decodable from layer 0** (97.5% acc) and remain flat throughout. AND-frac shows a **steep +0.53 rise** peaking at L3–L4.

**Implication:** AND-frac does NOT measure "when acoustic features appear." It measures a distinct phenomenon: the activation of the commitment circuit. Phoneme features exist from the start; the AND-gate is about *when the model commits to using them*.

This is a crucial clarification for Paper A's narrative: the Listen Layer (L*) is not where features are born, but where they are committed to a transcription path.
