## Proposed §4.8: Modality Collapse as gc(k) Suppression

### Core Definition

> **Modality Collapse onset** t\* = the first layer k > k_peak where gc(k) < ε (e.g., ε = 0.1 × gc_peak)

Equivalently: t\* = argmin_{k > k_peak} {gc(k)} gives the *depth at which audio information becomes causally negligible*.

Three gc(k) profile regions:
1. **Ramp** (0 → k_peak): audio info accumulates, gc(k) rising
2. **Peak** k_peak: maximum audio grounding — "Listen Layer"
3. **Collapse** (k_peak → L): gc(k) falls — model shifts to text-mode predictions

The collapse zone is what §4.8 quantifies.
