## M10 Summary (for events.jsonl)

**M10: Schelling Stability Score** — formalizes AudioSAE's empirical "50% stable features" finding:

```
SchellScore(F, S) = mean_{s'≠s} max_{F'∈SAE_s'} cos_sim(act_F, act_F')
Threshold: ≥ 0.85 = Schelling feature | 0.50-0.84 = semi-stable | <0.50 = unstable
```

**M10b: Causal Validation**
```
SchellCAC_correlation = Pearson r(SchellScore(F), CAC(F, A))
Prediction: r ≥ +0.5 → stable features are causally grounded
```

## Key theoretical contribution

M10 provides the first *principled* theory for why 50% of AudioSAE features are stable:
- Genuine Schelling (~35%): encode real phonological units → stable because structure is real
- Spurious Schelling (~15%): encode correlated acoustic surface forms → stable because
  co-occurrence patterns in training corpora are stable
- This is falsifiable with existing AudioSAE cross-seed checkpoints (Aparin et al.)
