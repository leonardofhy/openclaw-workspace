**arXiv:** 2402.17700

## Problem
Individual neurons are polysemantic (represent multiple concepts). How well can interpretability methods disentangle them?

## Method: RAVEL Benchmark
- **Core operation**: interchange interventions (activation patching)
- **Two metrics**:
  - **Cause(F, A)**: Does patching feature F → change model output for attribute A? (causal relevance)
  - **Isolate(F, A)**: Does patching F → NOT change output for other attributes A' ≠ A? (disentanglement)
- Best method: Multi-task DAS (MDAS) — learns distributed representations satisfying multiple causal criteria simultaneously
- Beats SAE, probes, PCA, DBM on disentanglement

## Results
- SAE features have better Isolate scores than raw neurons (monosemanticity helps isolation)
- MDAS achieves best Cause + Isolate trade-off
- Distributed representations > neuron-level analyses

## Connection to Q099 (our hypothesis)
**Hypothesis**: high-Isolate SAE features in MicroGPT = cross-seed Schelling-stable features

**Rationale**:
- High Isolate(F, A) means F represents A cleanly without bleeding into other attributes
  → This is a "pure" feature with a well-defined computational role
  → A feature with a clear computational role should be reliably discovered across random seeds
  → Therefore: high Isolate ↔ high Schelling stability
  
- Conversely, low-Isolate features are entangled (polysemantic) → different seeds may disentangle differently → low Schelling stability
