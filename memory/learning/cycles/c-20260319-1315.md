## Q113: Cascade Degree = 1 − AND_gate_fraction

### Claim
**Mechanistic definition:** `cascade_degree(L) = 1 − AND_gate_fraction(L)`

Where AND_gate_fraction(L) = fraction of active SAE features at layer L classified as AND-gates (require both audio AND text input to activate).

A high cascade_degree means text dominates → the model behaves like a cascade at layer L.
A low cascade_degree means audio is causally consulted → the model is "listening."

---

### Theoretical Grounding

**Cascade Equivalence Hypothesis (Billa 2026):** Speech LLMs behave like ASR+LLM cascades on text-sufficient tasks; under noise, cascades win (0dB: +7.6%). Gap #32 identifies that Billa has *no layer-level mechanistic account*.

**Paper A Claim 2 (to be stated):** gc(k) provides the first layer-by-layer mechanistic curve of the cascade transition. High gc(k) = audio causally drives output at layer k.

**Q113 unification:**
- gc(k) ↑ ↔ AND_gate_fraction(L) ↑ (already empirically confirmed: Pearson r = 0.9836, E1 batch results 2026-03-18)
- Therefore: `gc(k) ≈ AND_gate_fraction(L)` ↔ `cascade_degree(L) = 1 − gc(k)` (after normalization)
- This makes "cascade degree" a *feature-level* quantity: interpretable, computable without patching

**What this buys:**
1. **Mechanistic operationalization** of Billa's behavioral cascade equivalence → fills Gap #32
2. **Paper A Claim 2 upgrade:** "The Listen Layer is the depth at which cascade_degree is minimized = AND_gate_fraction is maximized"
3. **Predictive power:** cascade_degree can be computed via SAE feature inspection (cheaper than patching)

### Empirical Test Design (Mock)

**Setup:** MicroGPT mock (3-layer, hidden_dim=32, 16 SAE features/layer)

**Variables:**
- Per layer L: compute AND_gate_fraction(L) from Q089/Q091 AND-gate classifier output
- Per layer L: compute gc(L) from Q002/Q069 interchange intervention
- Compute cascade_degree(L) = 1 − AND_gate_fraction(L)
- Compute normalized_gc(L) = 1 − gc(L)/max_gc

**Expected result:**
- Pearson r(cascade_degree, 1−normalized_gc) > 0.8 across layers
- argmin(cascade_degree) = argmax(gc) = Listen Layer
- cascade_degree → 1.0 at late layers (full text collapse)

**Script skeleton:** `mock_cascade_degree.py`

```python
import numpy as np
from scipy.stats import pearsonr

# Mock data matching batch results 2026-03-18
layers = [0, 1, 2, 3, 4, 5]
and_fraction = [0.1, 0.3, 0.7, 1.0, 0.4, 0.1]  # peaks at gc peak (L3)
gc_k = [0.05, 0.15, 0.45, 0.56, 0.30, 0.10]    # from Q002 results

cascade_degree = [1 - f for f in and_fraction]
inverse_gc = [1 - g/max(gc_k) for g in gc_k]

r, p = pearsonr(cascade_degree, inverse_gc)
# Expected: r > 0.8, p < 0.01
listen_layer_and = np.argmin(cascade_degree)   # should match
listen_layer_gc  = np.argmax(gc_k)             # both = L3
assert listen_layer_and == listen_layer_gc, "Listen Layer mismatch!"
```
