## Interpretation

**Single-layer SAE captures ~38% of full-stack ΔGS signal**, but correlation is weak (r=0.54, p significant). This tells us:

1. ✓ **Listen-layer features ARE anomaly-sensitive** — consistent effect across random seeds
2. ⚠ **Single layer misses cross-layer interactions** — full-stack sees amplification across layers 3-5
3. 📊 **Ratio is stable** — 0.38 ratio holds across multiple training runs (suggests systematic relationship)

## Research Implications

### For T3 (gc × anomaly detection)

The weak correlation suggests that **gc(k) may operate at a different layer than where SAE-detectable anomalies peak**. Layer 3 peak ≠ Layer 0 peak. This connects to Paper A's finding that error incrimination peaks at mid-stack, not listen layer.

### For T5 (safety probes)

Single-layer ΔGS is **useful but incomplete**:
- ✓ Can be computed on CPU without model access
- ✓ Identifies which SAE features are anomaly-sensitive (top 5 features consistent across seeds)
- ⚠ Underestimates true sensitivity by ~62%
- ⚠ May miss cascading effects (L0 perturbation → L3 amplification)

**Practical:** Use single-layer ΔGS as a **fast screening metric** for feature importance, but validate critical findings with full-stack compute.
