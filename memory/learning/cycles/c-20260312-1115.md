## Problem

Q040 asks: "Produce Cause/Isolate scores under 2×2 emotion-speaker controls; summarize leakage pattern."

Emotion-Sensitive Neurons (ESNs) from Zhao et al. (2601.03115) are neurons in LALMs (Qwen2.5-Omni, Kimi-Audio) validated to causally carry emotion representation. But: **do they leak speaker identity?**

RAVEL (Huang et al., ACL 2024) provides the Cause/Isolate two-score framework for exactly this question.

---

## Method Synthesis: RAVEL applied to ESNs

### RAVEL Recap
- **Cause score**: when you patch neuron → does target attribute (emotion) change? (necessity test)  
- **Isolate score**: when you patch neuron → do *other* attributes (speaker ID) NOT change? (no-leakage test)  
- **RAVEL score** = harmonic mean(Cause, Isolate) — high only when both hold  
- SAEs in text fail on Isolate (features causally encode emotion but leak correlated attributes)

### 2×2 Emotion × Speaker Control Matrix

```
                  Speaker A          Speaker B
Emotion Happy  |  clip_HA         |  clip_HB         |
Emotion Angry  |  clip_AA         |  clip_AB         |
```

**Interchange intervention protocol** for Cause score (emotion):
- Source: clip_HA, Destination: clip_AA  
- Patch ESN activations from Source into Destination  
- Expected: destination output emotion = "happy" (Cause = 1)

**Interchange intervention protocol** for Isolate score (speaker ID):
- Source: clip_HA (Speaker A, Happy), Destination: clip_HB (Speaker B, Happy)  
- Patch ESN activations from Source into Destination  
- Expected: speaker ID should NOT change (= "still B" after patch)  
- If speaker ID flips → leakage detected → Isolate < 1

### Why ESNs likely leak speaker identity
1. Acoustic co-occurrence: happy speech has characteristic pitch/rate patterns that correlate with specific speaker styles
2. ESNs were validated via MAD/CAS selectors which don't control for speaker
3. Zhao et al. used within-speaker emotion classification — no speaker-leakage test was done
4. Prediction: **Cause ≈ 0.7-0.9, Isolate ≈ 0.3-0.6** → moderate RAVEL score
