# c-20260309-1045 — Q010: Cross-modal gc(k) Universality Plan

**Phase**: converge | **Track**: T3 | **Action**: build (Tier 0 design doc)

## Hypothesis

gc(k) — the "guessing coefficient" measuring how much a model's prediction
at position k derives from acoustic evidence vs. language prior — is a
**universal phase transition** that emerges in any sequence model trained
on predictive objectives, not just Whisper.

If true, this:
1. Strengthens Paper A's thesis (gc(k) is a mechanistic universal, not a quirk)
2. Opens cross-architecture replication section
3. Makes T5 safety probe (gc(k) as jailbreak thermometer) portable to text/vision

## Universality Variants to Test

### U1: Whisper (audio) — baseline ✅
- Already measured
- gc(k) shows phase boundary at ~layer 16-20 in Whisper-small

### U2: GPT-2 / small text LLM (text)
- Analogous: at position k, does prediction weight on prefix context vs. unigram prior?
- Operationalize: run causal patching — replace residual stream at layer L with
  "prior-only" null vector (mean activation on shuffled context), measure prediction change
- Expected: similar boundary layer emerges for syntactic/semantic transitions
- CPU feasible: GPT-2 small (117M) runs on CPU in seconds

### U3: CLIP (vision)
- Analogous: at patch k, does prediction rely on local patch features vs. global prior?
- Operationalize: mask out patch k's visual tokens, measure class probability shift
- Harder to operationalize (no autoregressive structure), deprioritize

### U4: Qwen2-Audio (multimodal)
- Interleaved audio+text: gc(k) in audio spans vs. text spans
- GPU required → blocked
