## Design

**Core hypothesis:**
- Clean audio → gc(k) peaks at layers 7-9 (DECAY shape, audio-dependent)
- FGSM adversarial → gc(k) collapses to layers 0-3 (early spike, audio structure destroyed)
- Detection rule: **t* < 4 → adversarial**

**Why this works mechanistically:**
- Whisper uses cross-attention between audio encoder states and decoder queries
- gc(k) = fraction of cross-attention heads that are "audio-dependent" (AND-gate) at layer k
- FGSM perturbations destroy audio-discriminative structure → model falls back to language priors (OR-gate / guessing mode)
- Early encoder layers carry low-level acoustic features; adversarial noise targets these → AND-gates never activate → t* shifts left

**Detection Protocol:**
1. Feed audio through Whisper encoder → probe gc(k) for each layer
2. Compute t* = argmax_k gc(k)
3. Flag as adversarial if t* < threshold (4 for Whisper-base)
