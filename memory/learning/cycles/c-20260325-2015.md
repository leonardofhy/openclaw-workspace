## Problem

Q174 wants linear probes at each encoder layer 0→L* for Whisper-base (6 layers).
Goal: show phoneme/word prediction accuracy PEAKS at L* (the AND-frac transition layer).
This would confirm L* is the "commitment boundary" — where audio representations crystallize
into discrete phoneme units before the decoder begins OR-gate text prediction.

---

## Probing 101 (applied to Whisper)

**What**: Train a logistic regression at each layer on frozen encoder hidden states.
Target: phone class label (IPA broad category) or word token ID.

**Why it confirms L***: If AND-frac theory is correct:
- Layers 0→L*: progressive phoneme crystallization → probing accuracy RISES
- At L*: maximum phoneme info → probing accuracy PEAKS
- After L*: representations start encoding text/language priors → phone-specificity DROPS

This "probe accuracy hill" corroborates AND-frac theory from a different angle (information-theoretic vs. causal).
