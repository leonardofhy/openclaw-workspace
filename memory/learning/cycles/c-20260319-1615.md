## Problem
RAVEL-style phoneme disentanglement metric for SAE features across K phoneme classes.

## Metric Design

**Bleed error** for feature `f_i` (tuned to peak class `c*`):
```
bleed(f_i) = (1/(K-1)) * Σ_{c≠c*} μ_c(f_i) / μ_{c*}(f_i)
```
- 0 = perfectly selective | 1 = fully entangled

**Disentanglement score**: `D(f_i) = 1 - bleed(f_i)`

**Per-class summary**: `D_class(c) = mean D(f_i) where c* = c`

## Expected Results
- Vowel features: D > 0.7 (clean)
- Stop features: D ≈ 0.5 (voiced/unvoiced bleed)
- D peaks at gc peak layer → connects to Paper A Claim 2

## Framework Connections
- High D → AND-gate (specific context required) — links Q109 → Q096
- Low D → OR-gate (broad phoneme response)
- bleed(f_i) inversely correlates with FAD phoneme specificity
