## Problem

How do we define "goodness" in a way that enables stable coordination across diverse intelligent agents with different values? Standard moral philosophy either requires a complete ethical theory (too strong) or permits total relativism (useless for alignment). Need a middle ground.

## Method / Key Concept

**Schelling goodness**: X is *Schelling-good* if, in a coordination game where a large diverse population of agents (from successful civilizations) must give the same binary verdict ("good" / "bad") about X — with no shared history except common knowledge of the question — they would converge on "good".

Core building blocks:
1. **Encouragement asymmetry**: calling a behavior "good" increases its probability; "bad" decreases it. (Empirical observation, not a definition of goodness.)
2. **Pro tanto morals**: "lying is bad" ≠ "never lie" — just a ceteris paribus strike against lying. Minimal, non-totalizing.
3. The framework avoids committing to any complete definition of good/bad while still being action-guiding.

Key distinction: "X is Schelling-good" ≠ "X is good". It's a claim about *what a large class of agents would say* in a constrained coordination game.
