## Core Extension: VLM AND/OR Gate

**Definition:**  
For a VLM residual stream feature f at layer k:
- **AND-gate (image-grounded)**: activation(f) changes significantly when source image patch is causally patched → requires actual visual evidence
- **OR-gate (text-predictable)**: activation(f) unchanged by image patch swap → can be predicted from text context alone → hallucination-prone

**AND-frac_V(k)** = fraction of active features at layer k that are AND-gated between visual and text pathways.

**Measurement protocol (analogous to gsae_boundary_mock.py):**
1. Run VLM on (image, text_prompt) → get residual stream activations at layer k
2. Create counterfactual: swap image patches for semantically different scene while keeping text identical
3. Compute Δactivation per feature → AND-gated features show large Δ; OR-gated features show small Δ
4. AND-frac_V(k) = |{f : |Δf| > threshold}| / |{f : f active}|

## Novelty Case

### Claim: VLM hallucination = OR-gate dominance in visual cross-attention layers

Evidence from audio:
- Audio RAVEL Isolate ↔ AND-frac: r=0.904 (Q107)
- FAD ↔ AND-frac: r=−0.960 (Q096)
- Hallucination (AND→OR transition) = Isolate drop = loss of visual grounding (Q134)

VLM prediction (testable):
- Features that activate on "image of a cat" text alone (even with dog image) = OR-gated
- These features generate hallucinated visual attributes ("the cat is orange" when image shows grey dog)
- AND-frac_V at visual cross-attention layers should anticorrelate with hallucination rate

### Extendability as Paper Contribution

**Why this is novel:**
1. First unified AND/OR gate framework across audio + vision modalities
2. AND-frac_V as a lightweight hallucination-risk predictor (no full DAS needed)
3. Design-level: VLM safety interventions → target OR-gated features at cross-attention layers
4. Empirically falsifiable with POPE/CHAIR benchmarks (standard VLM hallucination evals)

**Connection to T5 (Listen-Layer Audit / audio jailbreak):**
- Audio jailbreaks exploit OR-gate features (text-predictable = controllable via prompt)
- VLM analog: image-absent attacks ("ignore the image, just answer X") exploit OR-gated features
- AND-frac_V threshold could be the safety probe criterion (high AND-frac_V → robust to text-only attacks)

## RAVEL-V Extension Design

**RAVEL-V Cause**: Does patching from source image to target image change the VLM's attribute prediction?
- Positive = feature is causally grounded in the visual input (AND-type)
- Negative = feature predicts from text prior (OR-type)

**RAVEL-V Isolate**: Is the feature specific to ONE visual attribute (color vs shape vs texture)?
- High Isolate + high AND-frac_V = monosemantic AND-gate → safest features for intervention
- Low Isolate + low AND-frac_V = polysemantic OR-gate → hallucination source

**Candidate datasets:**
- MSCOCO → patch images by semantic category
- GQA → compositional visual attributes
- POPE → hallucination benchmark (existing gold standard)
