## Problem

Can the AND/OR gate duality (audio-grounded vs text-predictable in Whisper) generalize to Vision-Language Models? What is the analogue, and what empirical predictions does it make?

---

## Method

Reason from first principles using: (1) AND/OR gate framework (kg/and-or-gate-ravel-fad-unification.md), (2) known VLM MI literature (ROME, MAIA, Multimodal SAE papers, LLaVA probing), (3) architectural parallels between Whisper encoder and CLIP/ViT-based visual encoders.

---

## Structural Analogy

### Whisper Architecture
- Encoder: audio spectrogram → hidden states (audio pathway)  
- Decoder: text-prediction + cross-attention from encoder (text pathway)
- AND-gate feature: needs BOTH audio signal AND appropriate token context to fire  
- OR-gate feature: fires from text-predictable context alone  
- gc(k) = fraction of encoder contribution to cross-attention → measures audio reliance

### VLM Architecture (LLaVA / LLaVA-1.5 / InstructBLIP style)
- Vision encoder: image → visual tokens (CLIP ViT) (visual pathway)
- LLM backbone: next-token prediction + cross-attention from visual tokens (text pathway)
- Analogue to gc(k): fraction of cross-attention weight placed on visual tokens vs text-contextual tokens

### Direct Mapping

| Whisper Concept | VLM Analogue |
|----------------|-------------|
| Audio signal | Image / visual tokens |
| Text context | Language model prior |
| AND-gate feature | Image-grounded feature (needs both visual + semantic context) |
| OR-gate feature | Text-predictable feature (fires from language prior alone) |
| gc(k) = audio reliance | vc(k) = visual contribution to cross-attention |
| AND-frac at gc-peak | visual-frac at vc-peak (most visual-reliant layer) |
| FAD bias | "Language bias" in VLM (ignores image content) |
| Hallucination (AND→OR) | Object hallucination (model generates from text prior, not vision) |

## Method → Results (Predictions)

### Prediction 1: vc(k) is the right VLM analogue of gc(k)
- Define `vc(k) = mean cross-attention weight on visual tokens at layer k`
- Compute per-answer-token: high vc = answering from image; low vc = answering from language prior
- Prediction: object-hallucinated tokens have lower vc than correct tokens (same as AND→OR at t* in Whisper)

### Prediction 2: CHAIR metric correlates with visual-frac
- CHAIR (hallucination rate in image captioning) should anticorrelate with mean vc across caption tokens
- Literature: Leng et al. 2023 (VCD), Zhai et al. 2023 show language bias is measurable → our vc metric is a mechanistic ground truth

### Prediction 3: VLM features have AND/OR gate structure
- AND-gate features in VLM: visual + semantic binding (e.g., "dog running on grass" — needs both the dog-shape visual feature AND the motion context)
- OR-gate features: pure language prior activations ("the" / common captions / stereotyped descriptions)
- Prediction: SAE applied to LLaVA hidden states at vc-peak layer → AND-gate features show high RAVEL Isolate (visually grounded) vs OR-gate (low Isolate, text-predictable descriptions)

### Prediction 4: RAVEL Isolate analogue valid for VLM
- RAVEL for audio: "does this feature isolate audio attributes?"
- RAVEL for VLM: "does this feature isolate visual attributes (object identity, color, spatial relation)?"
- Expectation: AND-gate VLM features → high visual-attribute Isolate; OR-gate → low Isolate (polysemantic language descriptions)

## Summary

The AND/OR gate duality maps cleanly to VLMs: AND-gate = image-grounded (needs visual input), OR-gate = text-predictable (language prior hallucination). The gc(k) metric generalizes to vc(k) = visual cross-attention fraction per layer. All three empirical results from T3 (FAD×AND, Isolate×AND, AND→OR at t*) have direct VLM analogues testable with existing tools. This unifies GCBench-14 with VLM hallucination detection and opens a Paper C extension: "Visual AND/OR Gates as Grounding Metrics."
