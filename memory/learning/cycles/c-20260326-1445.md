## What Was Built

`q178_multilingual_andfrac.py` — Cross-lingual AND-frac experiment across 5 languages
(en/es/zh/ar/hi) × 2 groups (native/accented) on Whisper-base. Tests:
1. L*/L stability across script families (latin, logographic, arabic, devanagari)
2. English→other-language L* prediction accuracy
3. Accent gap Δ(L*/L) consistency across languages

## Key Results

| Lang | Native L*/L | Accented L*/L | Gap |
|------|------------|---------------|-----|
| en   | 0.783      | 0.690         | +0.093 |
| es   | 0.747      | 0.663         | +0.084 |
| zh   | 0.807      | 0.697         | +0.110 |
| ar   | 0.690      | 0.653         | +0.037 |
| hi   | 0.720      | 0.663         | +0.057 |

**L*/L stability:** Native σ = 0.042 (within tolerance); Accented σ = 0.017 (very stable)  
**Cross-lingual prediction:** en→es ✅ (err=0.036), en→zh ✅ (0.024), en→ar ⚠️ (0.093), en→hi ⚠️ (0.063)

## Novel Findings

1. **Universal listen-layer**: L*/L is consistent across ALL script families (0.69–0.81).
   Core thesis holds: AND-frac is NOT an English artifact.

2. **Accent gap is language-universal**: Δ(L*/L) > 0 for ALL 5 languages — accented speakers
   always trigger earlier commitment regardless of language. Core fairness claim strengthened.

3. **Morphological modulation** (unexpected): Arabic (high morphology) shows earliest commitment
   (L*/L=0.690) AND smallest accent gap (0.037). Hypothesis: Arabic training data in Whisper
   contains more accent diversity, or morphological richness reduces AND-gate sensitivity.
   → This is a real-data follow-up candidate (GPU, Whisper medium + Common Voice ar).

4. **Mandarin outlier**: zh has both highest native L*/L (0.807) AND largest accent gap (0.110).
   Tonal language hypothesis: Mandarin requires longer phonetic grounding → later commit.
