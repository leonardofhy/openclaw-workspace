# Q115 Design Doc: AudioSAE Stability Drop = t* Collapse
# Cycle: c-20260316-1515 | Phase: explore-fallback | Action: learn

## Hypothesis (Two-Instrument Unification)

**AudioSAE cross-seed stability drop at layer L* ≡ gc(k) collapse layer t***

Two independently-motivated instruments converge on the same mechanistic event:
- **Instrument A (gc(k))**: The decoder step t* where audio-grounded features stop causally
  influencing output — "the last step where the model truly listens."
- **Instrument B (AudioSAE stability)**: The Whisper/HuBERT encoder layer L* where
  cross-seed feature stability drops sharply — "the last layer where the model
  maintains disentangled audio representations."

**Claim**: t* (in decoding depth) and L* (in encoder layer depth) are dual signatures of
the same underlying phenomenon: the point where the model transitions from audio-grounded
to text-grounded inference.

## Evidence from AudioSAE (arXiv:2602.05027, EACL 2026)

| Finding | Relevance |
|---------|-----------|
| >50% features stable across 5 random seeds (Whisper + HuBERT) | Validates cross-seed stability as meaningful signal |
| Layer-wise stability curve exists (sharp drop at layers 6-7 in Whisper) | Suggests L* ≈ 6-7 for Whisper base |
| Feature steering reduces false speech detections by ~70% | Causal role of stable features confirmed |
| EEG–SAE feature correlation | Features are interpretable, not artefacts |

## Operationalization

### Step 1: Locate L* (AudioSAE instrument)
```
For each encoder layer l ∈ {0..N}:
    stability_score(l) = fraction of features with IIA > 0.5 across 5 seeds
L* = argmin(stability_score)   # or last layer before major drop
```

### Step 2: Locate t* (gc(k) instrument)
```
For each decoder step t:
    gc(k, t) = fraction of output affected by audio residual at step t
t* = argmin(gc(k, t))  # first step where gc drops below threshold θ
```

### Step 3: Test unification
- **Prediction A**: L*/N_enc ≈ t*/N_dec (normalized position alignment)
- **Prediction B**: Perturbations that shift L* also shift t* in the same direction
- **Prediction C**: AND-gate features (T3/Q096) cluster in layers ≤ L* 
- **Prediction D**: AudioSAE features from layers > L* have low causal incrimination (M9 signal)
