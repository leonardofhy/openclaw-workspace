## Paper Scanned

**arXiv:2509.09708** — *Beyond I'm Sorry, I Can't: Dissecting Large Language Model Refusal*  
Prakash et al., Sep 2025. Gemma-2-2B-IT + LLaMA-3.1-8B-IT.

### Problem
Internal causes of LLM refusal poorly understood. Want mechanistic jailbreak-critical feature set.

### Method
SAEs trained on residual-stream activations. 3-stage pipeline for finding refusal-critical SAE features:
1. **Refusal Direction**: find refusal-mediating direction in residual stream; collect SAE features near it
2. **Greedy Filtering**: prune to minimal causal set (ablation flips refusal → compliance)
3. **Interaction Discovery**: factorization machine (FM) captures nonlinear feature interactions in minimal set

### Results
- Pipeline yields interpretable jailbreak-critical feature sets
- **Redundant features** found: dormant normally, activate when earlier refusal features suppressed
- Fine-grained auditing + targeted intervention possible via SAE latent space manipulation
