## Paper: Discovering and Causally Validating Emotion-Sensitive Neurons in LALMs
Zhao, Schuller, Sisman (JHU + Imperial College). arXiv 2601.03115, Jan 2026.

### Problem
Do LALMs contain compact neuron subsets that causally encode emotion? Prior work: no causal validation at neuron level for affect in multimodal models.

### Method (log–identify–intervene)
1. **Log** SwiGLU gate activations on correctly-answered SER items (IEMOCAP, MELD, MSP-Podcast)
2. **Identify** ESNs via 4 selectors:
   - LAP: firing frequency P(e)
   - LAPE: entropy across emotions (lower = more selective)
   - MAD: magnitude contrast (M(e) vs avg other emotions) ← **best for causal isolation**
   - CAS: top-1 minus top-2 frequency margin ← **best for causal isolation**
3. **Intervene**: ablation (zero mask) or steering (scale ×(1+α))

### Key Results
- **MAD + CAS** produce strong self–cross ablation gap: ablating ESNs for emotion e disproportionately hurts e, not others
- **LAP + LAPE** insufficient — high firing ≠ causal specificity
- Targeted steering (amplify ESNs) biases predictions toward target emotion — actionable handle
- Label-free agnostic injection (2-Pass, Mix, Union) shows non-additive interference → ESN circuits are not independent

### Layer Locality
- ESNs cluster in **early (layer 0)**, **early-mid (6-8)**, **later (19-22)** decoder layers
- Non-uniform — NOT evenly distributed
- Partial cross-dataset transfer: some emotions transfer better than others

### Models tested
Qwen2.5-Omni-7B, Kimi-Audio, Audio Flamingo 3 — consistent findings across all three.

## Connection to Q040: Audio-RAVEL × ESN Leakage

**RAVEL framework** measures:
- **Cause**: editing the target attribute changes the output (causal relevance)
- **Isolate**: editing target attribute doesn't change other attributes (specificity / no leakage)

**Hypothesis for Q040:**
ESNs have high Cause for emotion but LOW Isolate because:
- Emotion co-occurs with speaker identity in SER corpora (angry speaker sounds different from calm speaker even phonetically)
- ESNs may encode a "emotional register" signal that conflates emotion + speaker gender/pitch + speaking rate
- Prediction: Cause ≈ 0.7, Isolate ≈ 0.2 under 2×2 emotion–speaker control

**Why leakage matters:**
- If ESNs leak speaker-gender info, ablating "anger ESNs" would also degrade recognition of male speakers in general
- This would be a confound — the neurons aren't truly emotion-specific
- RAVEL's categorical Cause/Isolate framework directly tests this

**Design for Q040 scorecard:**
1. Identify ESNs using MAD selector (best causal specificity per this paper)
2. Run RAVEL: edit emotion axis via ablation/steering
3. Track: does emotion acc drop (Cause)? Does gender/speaker acc also drop (Isolate failure)?
4. 2×2 control: {happy, angry} × {male, female} speaker
5. Expected result: Cause high across conditions; Isolate drops for male-skewed anger (angry-male more co-occurrent)
