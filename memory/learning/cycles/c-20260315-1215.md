## Papers Found

### 1. "Listen Like a Teacher: Mitigating Whisper Hallucinations using Adaptive Layer Attention and Knowledge Distillation"
- Tripathi et al., Nov 2025
- Key: Uses **adaptive layer attention** across decoder layers to mitigate hallucinations
- Implies: different decoder layers weight audio vs. LM prior differently → hallucination = layers tilting toward LM prior
- **gc(k) bridge**: gc(k) precisely measures this tilt per decoding step. "Listen Like a Teacher" identifies the layer axis; gc(k) identifies the *step* axis. Together = 2D hallucination map.

### 2. "Investigation of Whisper ASR Hallucinations Induced by Non-Speech Audio"
- Barański et al., Jan 2025
- Key: Non-speech audio (music, silence, noise) strongly induces hallucination → Whisper generates plausible text when audio signal is weak/ambiguous
- **gc(k) bridge**: weak audio signal → encoder provides low ground contribution → gc drops early → hallucination-prone segment begins

## Whisper Hallucination Taxonomy (from literature + prior knowledge)

| Type | Description | gc(k) prediction |
|------|-------------|-----------------|
| Repetition loop | "Thank you." repeated N times | gc collapses to ~0 mid-sequence → pure LM mode |
| Silence confabulation | Speech invented during quiet segment | gc near 0 from first token of hallucinated region |
| Language switch | Spurts of training-dominant language at unclear audio | Moderate gc with wrong phoneme alignment |
| Non-speech transcription | Music/noise → plausible-but-wrong text | gc drops below threshold at segment onset |

## Key Prediction for Q110

**Hypothesis**: `gc(k)` at decoding step `t` predicts hallucination onset:
- **Correct tokens**: gc(k) at/near peak → decoder actively using audio
- **Hallucinated tokens**: gc(k) << threshold → decoder disengaged from audio, running on LM prior
- **Collapse precedes error**: t* (collapse onset from Q085) predicts *which* token position will be wrong

**Novel contribution** (vs. "Listen Like a Teacher"):
- They fix hallucination via distillation + layer weighting (intervention)
- Q110 provides **mechanistic diagnosis**: gc(k) curve tells you *why* a specific token at step t was hallucinated
- gc-guided attention weighting = natural extension (future work)
