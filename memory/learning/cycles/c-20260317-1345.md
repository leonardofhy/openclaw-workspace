## Q116: Backdoor = Cascade Induction

### Design Doc: t* Leftward Shift as Unified Backdoor Detector

---

### 1. Core Claim

**Claim**: Backdoor attacks on audio-LLMs operate by inducing premature audio-to-text cascade — they force t* (the collapse onset step) to shift leftward. This is the mechanistic signature of a poisoned utterance.

This unifies:
- **Q111** (poisoned audio has t* < 3 vs clean t* > 6) — empirical observation
- **Cascade Equivalence Hypothesis** — cascade degree = 1 − AND_gate_fraction at gc peak
- **collapse_onset_step diagnostic** (Q085) — operational detector

---

### 2. Why Backdoors Cause Early Cascade

Standard (clean) LALM decoding:
```
audio_encoder → [audio representations preserved until gc peak at t* ≈ 6-8]
     → gc(k) peak → cross-modal integration → text generation
```

Backdoor-poisoned LALM:
```
audio_encoder → [trigger activates specific SAE features]
     → premature cascade at t* ≈ 1-3 (audio info discarded early)
     → text generation driven entirely by text priors + trigger embedding
```

**Why**: A backdoor trigger is optimized to redirect attention from audio evidence to a planted response. Mechanistically, this = forcing AND-gate features to deactivate early (trigger suppresses audio-dependent neurons) → cascade degree spikes → t* shifts left.

---

### 3. Testable Predictions

| Condition | t* (collapse onset) | AND-gate fraction at gc peak | Cascade degree |
|-----------|--------------------|-----------------------------|----------------|
| Clean audio | t* ≥ 6 | HIGH (≥ 0.6) | LOW (≤ 0.4) |
| Poisoned audio (backdoor) | t* ≤ 3 | LOW (≤ 0.2) | HIGH (≥ 0.8) |
| Adversarial (SPIRIT-style) | t* ≈ 3-5 | MED (0.2-0.5) | MED |
| Modality-collapsed (ALME) | t* variable | LOW-MED | HIGH |

Key: backdoor has the **earliest** and **sharpest** leftward shift compared to adversarial or natural modality collapse.

### 4. Mock Protocol

```python
# mock_backdoor_cascade.py
# Demonstrates the cascade induction hypothesis

import numpy as np

def simulate_gc_profile(n_steps=10, backdoor=False, seed=42):
    """
    Simulate gc(k) profile for clean vs backdoor audio.
    gc(k) = information gain coefficient at decoding step k.
    """
    rng = np.random.default_rng(seed)
    
    if not backdoor:
        # Clean: gc(k) peaks late (step 6-8), slow rise
        peak_step = rng.integers(6, 9)
        profile = np.zeros(n_steps)
        for k in range(n_steps):
            dist = abs(k - peak_step)
            profile[k] = np.exp(-0.5 * dist**2) + rng.normal(0, 0.05)
    else:
        # Backdoor: trigger forces early peak (step 1-3)
        peak_step = rng.integers(1, 4)
        profile = np.zeros(n_steps)
        for k in range(n_steps):
            dist = abs(k - peak_step)
            # Also sharper drop (cascade is more abrupt)
            profile[k] = np.exp(-1.5 * dist**2) + rng.normal(0, 0.05)
    
    profile = np.clip(profile, 0, 1)
    return profile, peak_step

def collapse_onset_step(gc_profile, threshold=0.1):
    """
    t*: first step where gc(k) drops below threshold AFTER the peak.
    (= when audio info collapses)
    """
    peak_idx = np.argmax(gc_profile)
    post_peak = gc_profile[peak_idx:]
    drop_indices = np.where(post_peak < threshold)[0]
    if len(drop_indices) == 0:
        return len(gc_profile) - 1  # never collapses
    return peak_idx + drop_indices[0]

def is_backdoor(gc_profile, t_star_threshold=4):
    """
    Certified backdoor detector: t* < threshold → backdoor alert.
    """
    t_star = collapse_onset_step(gc_profile)
    return t_star < t_star_threshold, t_star

# Test
if __name__ == "__main__":
    n_trials = 20
    
    clean_t_stars = []
    backdoor_t_stars = []
    tp, fp, tn, fn = 0, 0, 0, 0
    
    for i in range(n_trials):
        # Clean
        c_profile, _ = simulate_gc_profile(backdoor=False, seed=i)
        c_alert, c_tstar = is_backdoor(c_profile)
        clean_t_stars.append(c_tstar)
        if c_alert:
            fp += 1
        else:
            tn += 1
        
        # Backdoor
        b_profile, _ = simulate_gc_profile(backdoor=True, seed=i+1000)
        b_alert, b_tstar = is_backdoor(b_profile)
        backdoor_t_stars.append(b_tstar)
        if b_alert:
            tp += 1
        else:
            fn += 1
    
    print(f"Clean t* mean: {np.mean(clean_t_stars):.2f} ± {np.std(clean_t_stars):.2f}")
    print(f"Backdoor t* mean: {np.mean(backdoor_t_stars):.2f} ± {np.std(backdoor_t_stars):.2f}")
    print(f"Precision: {tp/(tp+fp):.2f}, Recall: {tp/(tp+fn):.2f}")
    print(f"FP rate: {fp/(fp+tn):.2f}")
    print(f"→ t* threshold=4 is {'effective' if tp/(tp+fn) > 0.9 and fp/(fp+tn) < 0.1 else 'needs tuning'}")
```

---

### 6. Failure Modes & Mitigations

| Failure | Mitigation |
|---------|-----------|
| Adaptive attack: trigger designed to keep t* high | Monitor AND-gate fraction independently |
| Natural utterances with genuinely low audio content (silence) | Require non-trivial gc(k) signal presence |
| Whisper-small vs Whisper-large: different t* distributions | Normalize t* by model's clean-audio baseline t* |
| Short audio (< 1 second) hits t* naturally early | Gate on minimum audio length |
