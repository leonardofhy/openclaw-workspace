## M9: Causal Abstraction Consistency (CAC)

### Motivation

Existing audio SAE metrics measure either:
- **Cause(F, A)**: does feature F cause attribute A to change? (output-direction, localization)
- **Isolate_out(F, A)**: does patching F leak to other attributes? (output-direction, collateral damage)

Neither metric asks: **does feature F's activation pattern match the causal structure of attribute A in the model?** A feature can score well on Cause + Isolate_out but still fail if:
1. It fires on spurious correlated inputs (input-direction leakage, Gap #35)
2. Its activation pattern is a proxy correlation, not a causal abstraction (Geiger et al. 2301.04709)

CAC tests both failure modes simultaneously.

---

### Formal Definition

**Setup:**
- F: SAE feature under test
- A: primary causal concept (e.g., voicing, jailbreak intent)
- B: spurious correlated concept (e.g., speaker gender, angry prosody)
- IIA(F, A): Interchange Intervention Accuracy of F on attribute A (Geiger et al. 2301.04709)
  - Measures alignment between F's activation pattern and the causal model of A
  - IIA = 1.0 → perfect causal abstraction; IIA = 0.5 → chance (random feature)

**Component metrics:**

```
Cause(F, A)     = P(ŷ_A flips | patch F → A-matched activation)
                  ∈ [0, 1]; higher = better localization

Isolate_out(F, A, B) = 1 − P(ŷ_B changes | patch F → A-matched activation)  [RAVEL style]
                       ∈ [0, 1]; higher = better output isolation

Isolate_in(F, A, B)  = 1 − max(0, R − 1)  where R = E[F | B=1, A=0] / E[F | B=0, A=0]
                       ∈ [0, 1]; higher = better input isolation (Directed Isolate, Gap #35)
                       R ≈ 1.0 → feature not triggered by B spuriously
                       R > 1.0 → input-direction leakage present

IIA(F, A)       = accuracy of attribute A classifier using F activations under
                  interchange interventions (source sample A=1, base sample A=0)
                  ∈ [0.5, 1]; higher = stronger causal abstraction alignment
```

**CAC score (M9):**

```
CAC(F, A, B) = IIA(F, A) × RAVEL_audio(F, A) × Isolate_in(F, A, B)

where RAVEL_audio(F, A) = harmonic_mean(Cause(F, A), Isolate_out(F, A))
```

**Interpretation:**
- CAC = 1.0: feature perfectly isolates A causally (in both directions) AND its activation pattern matches the causal hierarchy
- CAC = 0.0: at least one component fails — feature is either non-causal, leaky, or spuriously triggered
- CAC > 0.6: high-quality audio SAE feature (predicted threshold from RAVEL text baseline)

**Decomposition reveals failure mode:**

| IIA high | Cause high | Isolate_out high | Isolate_in high | Failure mode |
|----------|-----------|-----------------|-----------------|--------------|
| ✅ | ✅ | ✅ | ✅ | GOOD: genuine causal feature |
| ✅ | ✅ | ✅ | ❌ | Input leakage: B spuriously activates F |
| ✅ | ✅ | ❌ | ✅ | Output leakage: patching F damages concept B |
| ✅ | ❌ | ✅ | ✅ | Not causal: F aligns with A's structure but can't flip ŷ_A |
| ❌ | ✅ | ✅ | ✅ | Epiphenomenal: F causes behavior but doesn't match causal model |
| ❌ | ✅ | ❌ | ❌ | Spurious: F correlates with A but is neither causal nor isolated |

**Key insight**: CAC exposes the "consistently correlated but epiphenomenal" failure (AudioSAE Gap #22):
- AudioSAE's 50% stable features → high IIA + high Cause + high Isolate_out
- But if those features also fire on correlated non-cause B → low Isolate_in → low CAC
- CAC thus separates "SAE found a real causal feature" from "SAE found a stable proxy"

### Mock Example (Tier 0)

**Setup**: Whisper-small SAE, feature F_42, A = voiced phoneme (/b/, /d/, /g/), B = male speaker

**Mock activation tensors** (100 stimuli × 2 conditions):
```python
import numpy as np

np.random.seed(42)
n = 100

# 4-condition design (A × B)
# C1: A=1, B=1 (voiced + male)   E[F] = 0.85 (high, legitimate)
# C2: A=1, B=0 (voiced + female) E[F] = 0.80 (high, A causes F)
# C3: A=0, B=1 (unvoiced + male) E[F] = 0.45 (spurious input leakage!)
# C4: A=0, B=0 (unvoiced + female) E[F] = 0.12 (baseline)

F_C1 = np.random.normal(0.85, 0.10, n).clip(0, 1)
F_C2 = np.random.normal(0.80, 0.10, n).clip(0, 1)
F_C3 = np.random.normal(0.45, 0.12, n).clip(0, 1)  # leakage: B=male triggers F
F_C4 = np.random.normal(0.12, 0.08, n).clip(0, 1)

# Component computation
cause = 0.82         # P(ŷ_voiced flips | patch F → voiced activation) — high
isolate_out = 0.78   # 1 - P(ŷ_gender changes | patch F) — decent

R = np.mean(F_C3) / np.mean(F_C4)  # ≈ 0.45/0.12 ≈ 3.75
isolate_in = 1 - max(0, R - 1)     # = 1 - 2.75 = -1.75 → clipped to 0.0

iia = 0.84           # interchange intervention accuracy for voiced/unvoiced
ravel_audio = 2 / (1/cause + 1/isolate_out)  # harmonic mean = 0.799

cac = iia * ravel_audio * isolate_in
# = 0.84 × 0.799 × 0.0 = 0.0  ← FAILS due to input-direction leakage!

print(f"Cause={cause:.3f} | Isolate_out={isolate_out:.3f} | Isolate_in={isolate_in:.3f}")
print(f"IIA={iia:.3f} | RAVEL_audio={ravel_audio:.3f}")
print(f"CAC(F_42) = {cac:.3f}")
# Output: Cause=0.820 | Isolate_out=0.780 | Isolate_in=0.000
#         IIA=0.840 | RAVEL_audio=0.799
#         CAC(F_42) = 0.000
```

**Interpretation**: F_42 looks great under RAVEL (Cause=0.82, Isolate_out=0.78) and under IIA (0.84). These would all PASS existing metrics. But male speaker (B) spuriously activates F even when no voiced phoneme is present — a textbook example of acoustic co-occurrence bias. CAC=0.0 catches this; prior metrics would report F_42 as a high-quality feature.

**Corrected feature F_43** (no input leakage):
```python
F_C3_good = np.random.normal(0.11, 0.08, n).clip(0, 1)  # B=male doesn't trigger F
R_good = np.mean(F_C3_good) / np.mean(F_C4)  # ≈ 0.11/0.12 ≈ 0.92
isolate_in_good = 1 - max(0, R_good - 1)  # = 1 - 0.0 = 1.0 (no leakage)
cac_good = 0.84 * 0.799 * 1.0  # = 0.671 ← PASSES
print(f"CAC(F_43) = {cac_good:.3f}")  # 0.671 — high-quality feature
```

### M9: Causal Abstraction Consistency (CAC)

While Audio-RAVEL's RAVEL-audio score measures output-direction isolation, it does not test
whether spurious correlated inputs (e.g., speaker gender co-occurring with voiced phonemes)
trigger the feature independently of the target concept. We introduce M9: Causal Abstraction
Consistency (CAC), which multiplies three components:

  CAC(F, A, B) = IIA(F, A) × RAVEL_audio(F, A) × Isolate_in(F, A, B)

IIA (Geiger et al. 2301.04709) measures alignment between F's activation pattern and the
causal structure of attribute A. RAVEL_audio is the harmonic mean of Cause and Isolate_out
(§3.2). Isolate_in(F, A, B) = 1 − max(0, R−1) where R = E[F|B=1,A=0] / E[F|B=0,A=0]
tests whether spurious concept B triggers F when A is absent.

CAC = 1.0 iff the feature is a perfect causal abstraction: it precisely fires on A (not B),
its activation predicts A's causal role via interchange intervention, and patching it does not
collaterally affect B. CAC is the first metric to simultaneously test all three failure modes
of audio SAE features: spurious input activation, causal hierarchy mismatch, and output leakage.
