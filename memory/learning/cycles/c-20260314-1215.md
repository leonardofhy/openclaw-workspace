## Problem
RAVEL (ACL 2024, Huang et al.) evaluates whether interpretability methods can *disentangle* polysemantic representations. The two key criteria:
- **Cause**: Patching feature F for attribute A causally changes output about A (interchange intervention)
- **Isolate**: Patching F for attribute A does NOT change output for attribute B (disentanglement)

A method scores well only if it finds features that satisfy BOTH.

## Method
RAVEL uses activation patching (interchange interventions) across 5 entity types (city, person, verb, object, occupation), each with 4-6 attributes. Benchmark compares: neurons, PCA, probes, SAE, DAS, MDAS.
- **MDAS** (Multi-task DAS): jointly optimizes Cause + Isolate → SOTA. Shows distributed > neuron-level.
- SAEs help but don't fully solve isolation (polysemanticity survives into SAE features at neuron level).

## Results
- Neurons: high Cause (they do something) but poor Isolate (entangled)
- DAS/SAEs: better Isolate, especially with multi-task training
- Key finding: to get Isolate, you need to go beyond individual neurons to distributed representations

## Connection to Our Tracks (gc(k) × Schelling stability)

**Q099 hypothesis**: Schelling-stable features (cross-seed IIA-optimal) = isolated features?

**Answer**: Partial, with important nuance.

Stability and isolation are *related but distinct*:
- A Schelling-stable feature is one that SAE consistently recovers across random seeds → implies it captures a genuine structural direction in activation space
- But a dominant structural direction can be polysemantic (e.g., PC1 encodes multiple attributes simultaneously) → HIGH Cause, LOW Isolate
- Therefore: **stable ≠ isolated** in general, but stable features are *more likely* to be isolated than random features, because genuine monosemantic concepts recur across seeds while coincidental entanglements don't

**New prediction for gc(k) story:**
- Apply RAVEL-style Cause/Isolate metrics to audio-LLM layers relative to gc(k) peak (t*)
- Pre-t*: expect high Isolate (audio features isolated from language prior)
- At t*: minimum Isolate (maximum modality entanglement = convergence = gc peak)
- Post-t*: audio Cause drops to zero (collapse to language prior, audio features no longer causally mediating output)

This gives a **3-phase causal signature** across the decoding trajectory:
1. Isolate-high / Cause-low (audio not yet dominant)
2. Cause-high / Isolate-low (integration at gc peak)
3. Cause-zero (audio collapsed)

AND-gate features = high Cause + low Isolate (by design: they integrate two signals)
OR-gate features = high Cause + high Isolate (single-modality, monosemantic)
