## Problem
Can we isolate speaker identity vs linguistic content at L* in Whisper using RAVEL-style disentanglement?

## Method Summary (RAVEL, ACL 2024)
- **Core task:** Attribute disentanglement — find feature F that encodes attribute A *without* bleeding into other attributes
- **Metric:** Interchange interventions → two-score: **Cause** (patching F changes target attribute output) + **Isolate** (patching F doesn't affect other attributes)
- **SOTA method:** MDAS (Multi-task DAS) — finds distributed linear subspace features satisfying multiple causal criteria simultaneously
- **Dataset:** 5 entity types × multiple attributes, evaluated on Llama2-7B
- **Key finding:** Distributed features (linear subspaces of residual stream) beat neuron-level and SAE approaches

## Connection to AND-frac / L*
1. **Q204 Audio-RAVEL already aligned**: our (Cause, Isolate) two-score for AND-frac at L* is RAVEL-compatible
2. **Audio-RAVEL extension**: entities = audio segments; attributes = {speaker identity, content/phoneme, accent, language, emotion}
3. **L* as disentanglement layer**: hypothesis — L* is where Whisper separates *what was said* (content) from *who said it* (speaker identity) + acoustic style
4. **MDAS for Whisper**: apply multi-task DAS to Whisper encoder residual stream at L* to find content vs speaker subspaces
