### Synthesis: Gap #21 ‚Üí Track 1 Connection

**What Sadok et al. tells us:**
- SpeechTokenizer Layer 1 = semantic content (clean semantic signal)
- SpeechTokenizer Layers 2+ = acoustic details (pitch, identity, noise)
- This means we can SELECT which RVQ layers to corrupt with surgical precision

**What this enables for Track 1 (Audio IOI / Causal Benchmark):**

RVQ-layer-selective corruption = THE cleanest "clean/corrupt" signal design for audio patching:
- **Content-only corruption**: replace Layer 1 tokens from clean audio with different-content audio ‚Üí isolates semantic content pathway
- **Acoustic-only corruption**: replace Layers 2+ tokens ‚Üí isolates identity/pitch/acoustic pathway
- **Full replacement**: all RVQ layers (current naive approach = white noise = audio literature's suboptimal corruption flagged in Gap #6)
