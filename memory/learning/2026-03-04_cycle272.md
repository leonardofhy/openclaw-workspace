### Key formal insight for Paper A

**Graded faithfulness** (Geiger §3): A neural model N is a graded abstraction of causal model H if for intervention I on H, the corresponding intervention on N produces outputs that agree with probability p(I). 

For Paper A: gc(L) = p(audio-patching sensitivity | layer L) IS this graded faithfulness measure, applied to the causal model "model is grounded in audio at layer L." This makes gc(L) theoretically well-motivated, not just a heuristic metric.

**Linear representation hypothesis** (Geiger §4): Features are linearly encoded = alignment map is a linear projection. This is exactly what DAS finds. Paper A §2 can state: "We restrict alignment maps to linear projections following the linear representation hypothesis (Geiger et al. 2301.04709), which is necessary for non-trivial causal claims (Sutter et al. 2507.08802)."
