## Content

### Problem
SLMs (Qwen2-Audio-7B, LLaMa-Omni) are highly vulnerable to gradient-based adversarial jailbreaks.
- White-box PGD attack achieves **100% ASR** on bomb-making category against Qwen2Audio
- Avg ASR: 82.11% (Qwen2Audio) / 93.40% (LLaMa-Omni) with XTTSv2 female voice
- Key finding: underlying text LLMs are safe — attack success is entirely in the **audio modality**
- Audio perturbations are continuous-space (unlike discrete token manipulation in text) → larger attack surface

### Defense Mechanism (SPIRIT)
Three-stage activation patching defense:
1. **Identify noise-sensitive neurons**: compute ΔA_i = mean |A_i(x+δ) - A_i(x)| per neuron across sequence
2. **Select top-k% most sensitive neurons** for modification
3. **Apply targeted interventions**: 3 variants tested:
   - **Activation patching** (clean→adversarial substitution): inject clean or denoised audio activations to replace adversarial ones
   - **Bias addition**: add a learned bias toward "safe" direction in the identified neurons
   - **Neuron pruning**: ablate noise-sensitive neurons entirely

**Best result**: Activation patching with clean reference → 99% robustness improvement, negligible utility drop, no re-training.

### Key Technical Points
- Defense applied at **MLP layers of encoder or LM** (not attention; MLP = linear transformation → easier to intervene)
- "Denoised" reference = clean audio passed through the same model → activations substituted at noise-sensitive layers
- Target Qwen2-Audio-7B and LLaMa-Omni, both use Whisper encoder → Leo's infrastructure directly applies
- Code available: https://github.com/mbzuai-nlp/spirit-breaking.git
