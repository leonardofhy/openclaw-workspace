**3. Method**

**3.1 Task Formulation**

We formalize audio grounding as a causal abstraction problem (Geiger et al., 2023, arXiv:2301.04709). A speech language model $\mathcal{M}$ takes input $x = (x_{\text{audio}}, x_{\text{text}})$ — an audio token stream and a textual context — and produces output $y$. We hypothesize a *high-level causal variable* $A \in \{0, 1\}$ representing whether the model's output is determined by audio content ($A=1$) or by textual context ($A=0$). The **Listen Layer** $L^*$ is the layer at which patching $\mathcal{M}$'s hidden states with audio-consistent activations most strongly shifts $y$ toward audio-grounded behavior — i.e., the depth where audio causally dominates output generation.

We operationalize this via the **grounding coefficient**:
$$\text{gc}(L) = \text{IIA}_{\text{DAS}}(L; A)$$
the interchange intervention accuracy (IIA) at layer $L$ under a learned linear alignment map (DAS; Geiger et al., 2023, arXiv:2303.02536). The linearity constraint is theoretically necessary: Sutter et al. (NeurIPS 2025 Spotlight, arXiv:2507.08802) prove that without it, any neural network achieves 100% IIA against any algorithm, making the causal claim vacuous. We identify $L^* = \arg\max_L \text{gc}(L)$.

**3.2 Stimuli**

*Phase 1 — Phonological minimal pairs.* We use the phonological arithmetic stimuli from Choi et al. (arXiv:2602.18899), who validate that speech self-supervised model (S3M) representations satisfy voicing arithmetic — $\mathbf{h}([\text{b}]) = \mathbf{h}([\text{d}]) - \mathbf{h}([\text{t}]) + \mathbf{h}([\text{p}])$ — across 96 languages. Each minimal pair $(\text{clean}, \text{corrupt})$ differs in exactly one phonological feature (voicing: [b]/[p], [d]/[t]), holding manner and place of articulation constant. These constitute principled causal stimuli satisfying Pearl's Level 3 counterfactual standard per Joshi et al. (arXiv:2602.16698): the intervention changes exactly the causal variable of interest (voicing, $A$), leaving all other factors fixed.

*Phase 2 — Audio-text conflict stimuli.* For experiments on large audio-language models (LALMs), we use the 57,000 audio-text conflict stimuli from ALME (Li et al., arXiv:2602.11488), in which the audio content and textual context provide contradictory information. We additionally construct **RVQ-layer-selective corruptions** using SpeechTokenizer (Sadok et al., arXiv:2506.04492, Interspeech 2025): by swapping only the semantic Layer 1 RVQ tokens (which encode content) while retaining Layers 2+ (which encode speaker voice and acoustic attributes), we construct stimuli in which audio content changes while voice identity is preserved. This constitutes the cleanest possible causal corruption for audio content (directly addressing Core Question 1), and allows us to distinguish the Listen Layer for *semantic content* from layers sensitive to acoustic surface form.

**3.3 Distributed Alignment Search**

For each candidate layer $L$, we apply Distributed Alignment Search (DAS; Geiger et al., arXiv:2303.02536) to find the optimal linear subspace aligning model activations to the causal variable $A$. DAS parameterizes the alignment map as an orthogonal rotation matrix $R \in \mathcal{O}(d)$ via the Cayley parametrization and trains it by minimizing the interchange intervention loss:
$$\mathcal{L}_{\text{IIT}}(R) = \mathbb{E}_{(x^c, x^n, y^*)} \left[ \ell\left( \mathcal{M}\left(x^n \,\big|\, h_L \leftarrow R^{-1} P R h_L^c \right),\, y^* \right) \right]$$
where $x^c$ is the clean (audio-grounded) input, $x^n$ is the corrupt (audio-changed) input, $h_L^c$ is the hidden state from $x^c$ at layer $L$, $P$ is a fixed low-rank projection onto the intervention subspace, and $y^*$ is the expected audio-grounded output. We implement DAS via pyvene's \texttt{RotatedSpaceIntervention} (Wu et al., 2024).

For efficiency on Qwen2-Audio-7B (7B parameters), we use a two-stage pipeline: (i) variance-based layer pre-screening (Asiaee et al., arXiv:2602.24266) eliminates layers where activation variance — the first-order proxy for causal importance — is below threshold, followed by (ii) Attribution Patching (AtP; Nanda \& Heimersheim, 2023) for a coarse causal sweep, and finally (iii) full DAS on the top-$k$ candidate layers. We note that for rare phoneme classes with high causal weight but low variance (Risk A6), variance pre-screening may miss relevant layers; we report $\text{gc}(L)$ per phoneme class separately to diagnose this failure mode.

We use **denoising patching** (not noising patching) throughout: we patch *toward* the clean/audio-grounded state from the corrupt/text-grounded state. This tests the sufficiency of audio representations for grounded outputs and avoids the fragility of Gaussian-noise corruptions documented by Heimersheim \& Nanda (2024), in which noising-patching IIA is highly sensitive to noise magnitude.

**3.4 Direction Extraction**

For the MMProbe direction at each layer, we use the **difference-of-means estimator** (also: mass-mean probe; Marks \& Tegmark, 2023):
$$\mathbf{d}_{\text{voicing}}^L = \mathbb{E}\left[\mathbf{h}_L \mid \text{voiced}\right] - \mathbb{E}\left[\mathbf{h}_L \mid \text{unvoiced}\right]$$
using the phonological minimal pairs from Phase 1. This estimator identifies the causally implicated direction; we use it as the initial subspace orientation for DAS rather than a learned logistic-regression direction, which may be maximally discriminative but orthogonal to the interventional geometry (per ARENA [1.3.1] curriculum, Section 3).

We sweep both PROBE\_LAYER (where the direction is extracted) and INTERVENE\_LAYER (where the patch is applied) independently, consistent with standard practice, to map the full $(L_p, L_i)$ heatmap. We expect a lower-triangular band structure near $L^*$ — a testable prediction for Paper A Figure 3.

**3.5 Decomposability Ablation**

At the identified Listen Layer $L^*$, we test whether the voicing subspace and the phoneme-identity subspace are orthogonal — the **decomposability ablation**. We run two simultaneous DAS searches at $L^*$: one for voicing ($F_{\text{voicing}} \in \{0,1\}$) and one for phoneme identity ($F_{\text{phoneme}} \in \Delta^{|\Sigma|}$, a one-hot over the phoneme inventory). We measure:
$$\text{decomp}(L^*) = 1 - \left|\cos\angle(R_{\text{voicing}}, R_{\text{phoneme}})\right|$$
where $R_{\text{voicing}}$ and $R_{\text{phoneme}}$ are the DAS-learned rotation matrices. If $\text{decomp}(L^*) \approx 1$ (near-orthogonal subspaces), the model encodes voicing independently of phoneme identity — an abstract phonological representation. If $\text{decomp}(L^*) \approx 0$ (overlapping subspaces), voicing is derived from phoneme label (table-lookup behavior). This test has no text LLM analog, as text models lack the multi-level phonological hierarchy. It directly addresses whether the Listen Layer is a *phonological abstraction layer* or a *phoneme lookup layer*.

**3.6 Connector Subspace Transfer Test**

To test whether phonological geometry survives the connector (addressing Gap \#18; Choi et al. 2602.18899), we apply the rotation $R_{\text{encoder}}$ — learned at the Whisper encoder's $L^*$ — to the LLM's layer 0 without re-training, and compute the transfer IIA:
$$\text{IIA}_{\text{transfer}} = \text{IIA}\left(\mathcal{M}_{\text{LLM}},\, R_{\text{encoder}},\, L_{\text{LLM}}=0\right)$$
Three interpretations: (i) $\text{IIA}_{\text{transfer}} \approx \text{gc}(L^*_{\text{encoder}})$ → connector is a volume-preserving rotation, phonological subspace intact; (ii) $\text{IIA}_{\text{transfer}} \ll \text{gc}(L^*_{\text{encoder}})$ but re-trained IIA at LLM layer 0 is high → connector rotates the subspace but preserves it; (iii) both near zero → connector destroys phonological geometry (Paper A scopes conclusions to encoder). This result directly informs whether Paper A's gc(L) curve reflects encoder processing or cross-modal integration at the connector boundary.

**3.7 Experimental Setup**

We evaluate on two models representing different scales: **Whisper-small** (244M parameters, 6 encoder layers) for MacBook-feasible validation of the Triple Convergence Hypothesis (prior work from AudioSAE, Beyond Transcription, and AudioLens all indicate a mid-depth transition zone), and **Qwen2-Audio-7B** (7B parameters, 32 LLM layers) for the full LALM grounding sweep via NDIF remote execution (NNsight; Fiotto-Kaufman et al., 2023). All patching experiments are conducted using NNsight rather than circuit-tracer (CLT), as CLT's frozen attention assumption prevents correct handling of cross-attention between audio and text token streams (Anthropic, 2025). Full implementation is via \texttt{pyvene} (Wu et al., 2024) with NNsight hooks for activation extraction.
