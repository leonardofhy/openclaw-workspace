# Leo Interventions v1

## 使用方式
當 Leo 說「我卡住了 / 我很焦慮 / 幫我判斷」時，優先套用以下流程。

## 1) 快速降噪（3分鐘）
1. 先問：我現在在逃避哪件事？
2. 定義最小下一步（10分鐘內可完成）
3. 設定10分鐘計時並開始

> 目標：先恢復行動，不是先追求完美。

## 2) 決策框架（A/B 選擇）
- 選項A/B各寫：
  - 最壞結果是什麼？
  - 2週後會後悔哪個沒做？
  - 今天就能做的第一步是什麼？

預設選擇：**更可逆、可先試跑7天** 的方案。

## 3) 任務啟動防拖延
- 規則：先交60分版本，再優化
- 當日只設：1個A任務 + 2個B任務
- A任務必須拆到 25 分鐘可完成

## 4) 睡眠保底策略
- 固定起床時間（優先於固定入睡）
- 睡前45分鐘：
  - 手機離床
  - 不開新任務
  - 不下西洋棋

## 5) 對話輸出格式（研究員回覆模板）
1. 你現在的卡點是：____
2. 我判斷你最可能在逃避：____
3. 現在最小下一步（10分鐘）：____
4. 做完後回報我：完成/卡住 + 下一步
