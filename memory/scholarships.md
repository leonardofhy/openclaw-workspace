# Leo 獎學金清單（2026年2月整理）

> 身分：台大電信所碩一 / 僑生 / 馬來西亞公民
> 最後更新：2026-02-17

## 🟢 現在可行動

### 僑委會 受理捐贈僑生獎助學金
- 金額：視項目（數萬～數十萬）
- 資格：在學僑生、上年成績均75分以上
- 申請：第二學期開學後，向台大 OIA 提交
- 連結：https://scholarship.ocac.gov.tw/

---

## 📅 按月份排序

### 4月
- **台大 國際傑出研究生獎學金** ⭐
  - 金額：學費減免 + $8,000/月（工程學院）
  - 資格：碩士2年以下，僑生可申請
  - 連結：https://isss.ntu.edu.tw/finances/scholarships/

- **電信所 B/E 類獎勵金** ⭐
  - 金額：數千～萬元/學期
  - 資格：電信所碩博士生，不限國籍
  - 備註：每學期都有，必申請

### 7-8月
- **中技社 境外生研究獎學金** ⭐
  - 金額：$150,000（一次性）
  - 資格：境外生理工碩博士，需系所推薦
  - 連結：https://www.ctci.org.tw/8838/talent/ctci-scholarship/

- **中技社 境外生生活助學金**
  - 金額：$100,000（一次性）

- **科林論文獎及傑出科技獎學金**
  - 金額：數萬元
  - 資格：台大碩博士生，需優秀論文

- **電資院研究生校長獎**
  - 金額：獎金 + 榮譽
  - 資格：電資學院碩博士生，不限國籍

### 8-9月
- **潘文淵文教基金會獎學金** ⭐
  - 金額：約 $100,000
  - 資格：電資相關碩博士生，過去不限國籍

- **樹本教育基金會獎學金**
  - 金額：$30,000-50,000
  - 資格：理工科研究生，僑生需確認

### 9-10月
- **Norman and Lina Chang Foundation 獎學金** ⭐
  - 金額：數萬元
  - 資格：電機相關研究生，海外華人基金會，僑生友善

- **故何君先生紀念獎學金**
  - 金額：數萬元
  - 資格：電機相關研究生，僑生需確認

- **錢思亮先生紀念獎學金**
  - 金額：數萬元
  - 資格：台大理工研究生，僑生需確認

- **台北天母扶輪社獎學金**
  - 金額：$50,000

- **中華扶輪教育基金**
  - 資格：研究生，各校限推薦2名

### 10-11月
- **鄧立登清寒獎助學金**
  - 金額：數萬元
  - 資格：清寒研究生，僑生需確認

- **TSIA 台灣半導體產業協會獎**
  - 金額：$100,000-200,000
  - 資格：半導體相關研究碩博士生

- **1975級電機系友科技研究創新獎勵**
  - 金額：數萬元
  - 資格：電機相關碩博士生

### 11月（碩二後）
- **感恩與傳承獎學金**
  - 資格：碩博士2年級以上

### 隨時
- **電信所 TA (A類獎勵金)**
  - 金額：$6,000-8,000/月
  - 資格：電信所碩博士生，不限國籍

- **電信所國際會議補助**
  - 金額：數千～數萬
  - 資格：有論文發表時申請，每2-3月一輪，不限國籍
