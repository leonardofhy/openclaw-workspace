## 🇨🇳 深圳行程建議（3/7 晚-3/8）

### 華強北（重頭戲！MacBook 升級）

**📍 地址**：福田區華強北路（地鐵：華強北站）

#### MacBook Air SSD 升級

**你需要準備的：**
1. **M.2 NVMe 轉接板**（adapter board）— 華強北有賣，¥30-50
   - MacBook Air (M1/M2) 用的是 Apple 專有 SSD 接口，需要轉接板才能裝標準 M.2 NVMe
   - 搜索關鍵字：「MacBook Air M1 SSD 轉接板」或「Apple SSD adapter M.2」
2. **M.2 NVMe SSD** — 建議 1TB 或 2TB
   - 華強北價格參考：1TB ~¥350-500，2TB ~¥600-900
   - 推薦品牌：西數 SN770、三星 980 PRO、致態 TiPlus7100（國產性價比高）
3. **工具**：
   - **五角螺絲刀**（Pentalobe P5）— 開底殼用
   - **六角星螺絲刀**（Torx T5）— 拆內部螺絲用
   - 華強北工具店都有，一套 ¥10-20

**流程：**
1. 先備份（Time Machine 或其他方式）— **出發前在台灣做好！**
2. 到華強北買轉接板 + SSD + 工具
3. 可以請店家幫你裝（通常加 ¥50-100 工費），或自己裝
4. 裝好後用 USB 隨身碟安裝 macOS（或從網路恢復）

**⚠️ 風險提醒：**
- 確認你的 MacBook 型號對應的轉接板型號（M1 和 M2 可能不同）
- 第三方 SSD 可能不支援 macOS 的 S4 休眠（sleep），影響不大但要知道
- 保修會失效（但 SSD 升級是可逆的，換回去就行）

#### MacBook 鍵帽更換

**方案：**
- **整套鍵帽**：淘寶 ¥85 / 閒魚 ¥20-40（二手）
- 華強北現場：部分店家有散賣鍵帽，但完整套可能要問
- MacBook Air 用的是**剪刀腳結構**（Scissor mechanism），更換方法：
  1. 用塑料撬棒從鍵帽邊緣輕輕撬起
  2. 取下舊鍵帽（注意不要弄壞下面的剪刀腳支架）
  3. 對準卡扣按壓新鍵帽直到「咔嗒」一聲

**建議**：如果只是個別鍵帽壞了，華強北買散的划算；如果要換整套，提前淘寶下單寄到深圳住處更穩

### 華強北其他好逛的

- **賽格大廈**（SEG Plaza）— 電子元件、手機配件、各種數碼產品。1-4 樓最熱鬧
- **華強電子世界** — 比賽格更零售導向，手機殼、充電器等
- **明通數碼城** — 二手手機、翻新機、各種奇葩數碼產品
- **遠望數碼商城** — 手機維修聚集地，這裡問 MacBook 維修/升級最專業

### 深圳其他推薦

**美食：**
- **海底撈**（華強北附近多家）— 如果想吃火鍋，服務一流
- **太二酸菜魚** — 深圳本土品牌，酸菜魚做得好
- **文和友** — 南山區，沉浸式餐飲體驗（復古風）
- **喜茶**（HEYTEA）— 深圳起家的奶茶品牌，必喝
- 華強北周邊小吃街：腸粉、煲仔飯、砂鍋粥

**景點（如果有空）：**
- **南頭古城** — 深圳文創聚集地，咖啡店 + 獨立書店 + 藝術空間
- **深圳灣公園** — 海邊散步看對岸香港
- **中心書城** — 亞洲最大書店之一，值得逛

### 深圳支付
- 微信支付 / 支付寶（提前綁好）
- 部分店家收現金人民幣
- Visa/Mastercard 在大商場可用，小店基本不行

## 🚇 交通指南

### 香港 → 深圳（過境）

**推薦：高鐵 / 東鐵線**
1. **高鐵**（西九龍站 → 福田站）：14 分鐘，HKD 78，最快但要提前買票
2. **東鐵線**（紅磡/上水 → 羅湖）：需過關，慢但班次多
3. **福田口岸**：東鐵線落馬洲站，連接深圳福田（離華強北近）

**從你的酒店（上環）出發：**
- 上環 → 港鐵到中環 → 換東鐵線到落馬洲 → 過關到福田（約 1.5 小時）
- 或：上環 → 港鐵到西九龍 → 高鐵到福田（約 1 小時，但貴）

### 深圳 → 香港（回程同理）

### 香港內部
- **八達通**（Octopus）：港鐵 + 巴士 + 便利店都能用，機場 or 便利店買
- 或直接用 Apple Pay / Google Pay 感應入閘
- 上環站是港島線，去哪都方便
