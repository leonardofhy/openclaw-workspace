### 🔧 華強北 MacBook SSD 升級攻略

**路線：先去遠望 → 再逛賽格**

**第一站（09:30）：遠望數碼商城**（華強北路 1002 號）
- MacBook 維修/升級最集中的地方
- **至少問 3 家比價**，說「MacBook Air SSD 升級，要轉接板 + SSD」
- 購買清單：

| 物品 | 參考價 | 備註 |
|------|--------|------|
| M.2 NVMe SSD 1TB | ¥350-500 | 推薦：致態 TiPlus7100（性價比）或西數 SN770 |
| M.2 NVMe SSD 2TB | ¥600-900 | 預算夠就上 2TB |
| MacBook 專用轉接板 | ¥30-50 | 告訴店家你的 MacBook 型號！ |
| Pentalobe P5 + Torx T5 螺絲刀 | ¥10-20 | 一套搞定 |
| 鍵帽（整套） | ¥50-85 | 問有沒有你的型號 |
| 鍵帽（單顆） | ¥5-10/顆 | 只壞幾顆就買散的 |

- **推薦請店家當場裝**（加 ¥50-100 工費）：省心、有問題當場解決、店家有專業工具
- 如果自己裝：找個安靜咖啡廳，30-45 分鐘搞定

**殺價技巧：**
- 開價通常 1.5-2 倍，目標砍到 6-7 折
- 「別家報 XXX 塊」是最好的武器（所以先問 3 家）
- 態度友善但堅定，不急著掏錢
- SSD 品牌貨（三星、西數）砍價空間小；國產（致態、長江存儲）空間大

**第二站（11:00）：自由逛**
- **賽格大廈**（SEG Plaza）1-4 樓 — 電子元件天堂
- **華強電子世界** — 手機殼、充電器、配件
- **明通數碼城** — 二手手機、翻新機、各種奇葩產品
