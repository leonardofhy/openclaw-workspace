# Workspace Rules

- **Default Root**: `/Users/leonardo/Workspace`
- **Sub-folders**: 
  - `memory/` -> `memory/` (symlink or copy)
  - `skills/` -> `skills/` (symlink or copy)
  - `scripts/` -> `scripts/` (symlink or copy)

- **Agent Behavior**: 
  - Always default to creating new projects in `/Users/leonardo/Workspace/{project_name}`.
  - For simple scripts, use `/Users/leonardo/Workspace/scripts`.
