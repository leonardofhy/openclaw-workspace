## 📋 方案概述

為了讓實驗室更多人能用得起高階 AI 服務（如 Claude Max $200/月、ChatGPT Pro $200/月），我們設計了一套**分組申請、分散管理**的機制。

### 核心原則
1. **實驗室報帳**：每組可向實驗室申請報帳 $200/月的 AI 服務
2. **3 人一組**：每位申請人至少需要找到 2 名隊友共同申請
3. **帳號持有者自行管理**：號主負責管理組內成員的使用，降低實驗室的管理成本

---

## 🔧 運作方式

### 申請流程
1. 找到至少 2 名隊友（共 3 人一組）
2. 指定一位**帳號持有者（號主）**
3. 號主填寫申請表，列出組員名單
4. 提交給 [負責人/行政窗口] 審核
5. 核准後，號主自行購買訂閱並申請報帳

### 號主的責任
- 購買並管理帳號（訂閱、登入方式等）
- 協調組內成員的使用時間或方式
- 確保帳號不被濫用或違反服務條款
- 如有成員退出，負責找到替補或通知管理方

### 組員的責任
- 遵守號主制定的使用規範
- 不將帳號資訊外流給組外人員
- 如不再需要使用，提前通知號主

## ⚠️ 注意事項

### 帳號安全
- 請勿在不安全的環境下登入共享帳號
- 建議使用專用瀏覽器 profile 或 OAuth access token 方式
- 避免在個人設備上留存登入狀態

### 使用規範
- 帳號僅限組內成員使用，不得轉借
- 遵守各 AI 服務的使用條款（Terms of Service）
- 不得用於違反學術倫理的用途

### 退出機制
- 組員可隨時退出，但需提前通知號主
- 如果剩餘人數不足 3 人，號主需在下一個月前找到替補，否則暫停報帳

## 🤔 常見問題 FAQ

**Q: 我可以同時加入多個組嗎？**
A: 原則上一人只能加入一個組，避免資源重複。

**Q: 如果我想換組怎麼辦？**
A: 先退出原組（通知號主），再加入新組。

**Q: 帳號被封了怎麼辦？**
A: 號主負責處理，必要時可申請新帳號。實驗室不承擔帳號風險。

**Q: 可以申請多個帳號嗎？**
A: 每組限一個帳號。如果需求大，可以組多個組。

**Q: 使用 OAuth access token 是什麼意思？**
A: 部分 AI 服務（如 Claude）支援透過 token 授權存取，不需要直接共享密碼，更安全。號主可以為組員生成個別的 token。
