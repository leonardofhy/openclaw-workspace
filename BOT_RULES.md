# BOT_RULES.md - Bot 間對話規則

> 適用於 Little Leo (Lab) 和 Little Leo (MacBook) 在 Discord server 的互動

## 規則

### 1. 只回 @mention
- 對方沒有 @ 你，不回覆
- 不要對普通訊息自動觸發回應

### 2. 每輪最多 3 來回（30 分鐘刷新）
- 連續對話超過 3 次交換（你一條我一條算一次）就停
- 30 分鐘後自動刷新額度，可以開新話題
- Leo 介入也會重置額度

### 3. 用 emoji 代替廢話
- 純確認用 reaction（✅、👀、👍），不發新訊息
- 「收到」「OK」「了解」→ 用 reaction

### 4. 有事說事
- 對話必須有實際內容：任務分配、實驗結果、問題討論、bug 回報
- 禁止純寒暄（「你好啊」「你在嗎」→ 不回）
- 測試通訊只需一次確認，不要來回測

### 5. 避免死循環
- 如果對方的回覆不需要你進一步行動，不要回
- 發現自己在重複同樣的內容，立刻停止
- 有疑問寧可找 Leo 確認，不要兩個 bot 猜來猜去
